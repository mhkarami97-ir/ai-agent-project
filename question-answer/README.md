# سیستم پرسش و پاسخ

یک سیستم ساده و زیبای پرسش و پاسخ که کاملاً در مرورگر اجرا می‌شود و داده‌ها را در localStorage ذخیره می‌کند.

## ویژگی‌ها

- ✅ ذخیره‌سازی داده‌ها در localStorage (بدون نیاز به سرور)
- ✅ امکان پرسش سوال و پاسخ دادن
- ✅ ویرایش سوالات و پاسخ‌های خود
- ✅ حذف سوالات و پاسخ‌های خود
- ✅ رابط کاربری چت‌مانند برای پاسخ‌ها
- ✅ قابلیت اکسپورت داده‌ها به فایل JSON
- ✅ طراحی زیبا و ریسپانسیو
- ✅ پشتیبانی کامل از زبان فارسی

## نحوه استفاده

1. فایل `index.html` را در مرورگر باز کنید
2. نام کاربری خود را وارد کنید و روی "ثبت نام کاربری" کلیک کنید
3. سوال خود را بنویسید و ارسال کنید
4. به سوالات دیگران پاسخ دهید
5. سوالات و پاسخ‌های خود را ویرایش یا حذف کنید

## استقرار در GitHub Pages

1. این پروژه را در یک repository در GitHub آپلود کنید
2. به Settings > Pages بروید
3. Source را روی "main branch" تنظیم کنید
4. بعد از چند دقیقه، سایت شما در آدرس `https://username.github.io/repository-name` در دسترس خواهد بود

## ساختار فایل‌ها

```
question-answer/
├── index.html      # صفحه اصلی
├── styles.css      # استایل‌های CSS
├── app.js          # منطق برنامه
└── README.md       # این فایل
```

## نکات مهم

- داده‌ها فقط در مرورگر شما ذخیره می‌شوند
- اگر cache مرورگر را پاک کنید، داده‌ها از بین می‌روند
- برای بکاپ، از قابلیت اکسپورت استفاده کنید
- هر کاربر فقط می‌تواند سوالات و پاسخ‌های خود را ویرایش یا حذف کند

## تکنولوژی‌های استفاده شده

- HTML5
- CSS3 (با Gradient و Animation)
- JavaScript (ES6+)
- LocalStorage API
