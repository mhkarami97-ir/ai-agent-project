## 🚀 مراحل استقرار

```bash
# 1. Initialize Git
git init

# 2. Add files
git add .

# 3. Commit
git commit -m "Initial commit: Typing practice app"

# 4. Add remote
git remote add origin https://github.com/USERNAME/REPO-NAME.git

# 5. Push
git branch -M main
git push -u origin main
```

## ⚠️ قبل از Push

- [ ] `USERNAME` را با نام کاربری خود جایگزین کرده‌اید
- [ ] `REPO-NAME` را با نام مخزن خود جایگزین کرده‌اید
- [ ] مطمئن هستید تمام فایل‌ها آماده هستند

## 📞 در صورت مشکل

اگر مشکلی پیش آمد:

1. **خطای 404**: 
   - مطمئن شوید `index.html` در ریشه است
   - منتظر 5 دقیقه برای build بمانید

2. **استایل‌ها نمایش داده نمی‌شوند**:
   - مسیرهای CSS را بررسی کنید
   - مطمئن شوید پوشه `assets` وجود دارد

3. **جاوااسکریپت کار نمی‌کند**:
   - Console مرورگر را بررسی کنید (F12)
   - مسیر `assets/script.js` را چک کنید

4. **فونت نمایش داده نمی‌شود**:
   - اتصال اینترنت را بررسی کنید
   - لینک Google Fonts را چک کنید
