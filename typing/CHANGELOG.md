# تاریخچه تغییرات

## نسخه 1.0.0 (2025-01-20)

### ✨ ویژگی‌های جدید
- پشتیبانی کامل از دو زبان فارسی و انگلیسی
- 5 متن مختلف برای تمرین در هر زبان
- محاسبه WPM (کلمه در دقیقه)
- محاسبه درصد دقت تایپ
- تایمر زمان تمرین
- نمایش کاراکتر مورد انتظار در کادر بزرگ
- رنگ‌بندی تایپ صحیح (سبز) و نادرست (قرمز)
- ذخیره خودکار تاریخچه تمرین‌ها (آخرین 20 مورد)
- نمودار پیشرفت با Canvas API
- صفحه تاریخچه تمرین‌ها با جزئیات کامل

### 🎨 طراحی
- رابط کاربری زیبا با گرادیانت بنفش
- فونت Vazirmatn از Google Fonts
- پشتیبانی کامل RTL برای فارسی
- انیمیشن‌های نرم برای تمام المان‌ها
- اثرات بصری برای بازخورد تایپ
- طراحی کامل رسپانسیو (موبایل، تبلت، دسکتاپ)
- آیکون SVG سفارشی

### 🌐 ویژگی‌های وب
- کاملاً سمت کاربر (Client-side only)
- بدون نیاز به سرور یا بکند
- ذخیره‌سازی با LocalStorage API
- آماده برای استقرار در GitHub Pages
- سازگار با تمام مرورگرهای مدرن

### 📝 مستندات
- راهنمای کامل فارسی (README.md)
- راهنمای کامل انگلیسی (README.en.md)
- دستورالعمل استقرار (DEPLOYMENT.md)
- مرجع دستورات Git (GIT_COMMANDS.md)
- راهنمای شروع سریع (START.md)
- راهنمای استفاده (USAGE_GUIDE.md)
- خلاصه پروژه (PROJECT_SUMMARY.md)
- لایسنس MIT (LICENSE)

### 🛠️ تکنیکال
- HTML5 معتبر
- CSS3 با Flexbox و Grid
- JavaScript Vanilla (بدون فریمورک)
- Canvas API برای رسم نمودار
- LocalStorage برای ذخیره‌سازی
- Event-driven architecture
- کد تمیز و مستندسازی شده

### 📱 رسپانسیو
- بهینه‌سازی برای موبایل (< 480px)
- بهینه‌سازی برای تبلت (480px - 768px)
- بهینه‌سازی برای دسکتاپ (> 768px)
- تست شده در تمام اندازه‌های صفحه
