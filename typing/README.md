# تمرین تایپ فارسی/انگلیسی

یک برنامه وب ساده و کاربردی برای تمرین و بهبود مهارت تایپ به زبان‌های فارسی و انگلیسی.

## ویژگی‌ها

- ✨ **رابط کاربری زیبا و مدرن**: طراحی تمیز با گرادیانت‌های جذاب
- 🌐 **پشتیبانی دو زبانه**: امکان تمرین به فارسی و انگلیسی
- 📊 **محاسبه سرعت تایپ**: نمایش WPM (کلمه در دقیقه)
- 🎯 **محاسبه دقت**: درصد دقت تایپ شما
- ⏱️ **تایمر**: اندازه‌گیری زمان تمرین
- 📈 **نمودار پیشرفت**: نمایش گرافیکی پیشرفت تایپ
- 📝 **متن‌های متنوع**: ۵ متن مختلف برای هر زبان
- 💾 **ذخیره تاریخچه**: ذخیره خودکار نتایج تمرین‌ها
- 📱 **طراحی ریسپانسیو**: سازگار با موبایل، تبلت و دسکتاپ
- 🎨 **فونت زیبا**: استفاده از فونت Vazirmatn

## نحوه استفاده

1. زبان مورد نظر خود را انتخاب کنید (فارسی یا انگلیسی)
2. متن دلخواه را از منوی کشویی انتخاب کنید
3. شروع به تایپ کنید - تایمر به صورت خودکار شروع می‌شود
4. کاراکترهای صحیح به رنگ سبز و اشتباه به رنگ قرمز نمایش داده می‌شوند
5. پس از اتمام، نتایج شما ذخیره می‌شود
6. برای مشاهده تاریخچه و پیشرفت خود، دکمه "مشاهده تاریخچه" را بزنید

## تکنولوژی‌ها

- HTML5
- CSS3 (با Flexbox و Grid)
- JavaScript (Vanilla JS)
- LocalStorage برای ذخیره‌سازی داده‌ها
- Canvas API برای رسم نمودار
- Google Fonts (Vazirmatn)

## نصب و راه‌اندازی

### روش ۱: مشاهده مستقیم فایل

فایل `index.html` را در مرورگر خود باز کنید.

### روش ۲: استفاده از سرور محلی

```bash
# با استفاده از Python
python -m http.server 8000

# یا با Node.js
npx serve
```

سپس به آدرس `http://localhost:8000` بروید.

### روش ۳: استقرار در GitHub Pages

1. مخزن را در GitHub ایجاد کنید
2. فایل‌ها را آپلود کنید
3. به تنظیمات مخزن بروید
4. در بخش Pages، برنچ `main` و پوشه `/root` را انتخاب کنید
5. لینک سایت شما آماده است!
