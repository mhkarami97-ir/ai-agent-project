## جریان کار

### 1️⃣ انتخاب زبان
```
کلیک روی [فارسی] یا [English]
→ صفحه به RTL یا LTR تغییر می‌کند
→ متن‌ها به زبان انتخابی نمایش داده می‌شوند
```

### 2️⃣ انتخاب متن
```
کلیک روی منوی کشویی
→ انتخاب یکی از 5 متن
→ متن در کادر بالا نمایش داده می‌شود
```

### 3️⃣ شروع تایپ
```
کلیک در کادر تایپ
→ شروع به تایپ متن
→ تایمر شروع می‌شود
→ کاراکترها رنگی می‌شوند:
   ✓ سبز = صحیح
   ✗ قرمز = اشتباه
   ▶ زرد = کاراکتر فعلی
```

### 4️⃣ مشاهده نتایج
```
در حین تایپ:
→ WPM به‌روزرسانی می‌شود
→ درصد دقت محاسبه می‌شود
→ تایمر در حال اجرا است
```

### 5️⃣ اتمام تمرین
```
تایپ کامل متن
→ پیام تبریک نمایش داده می‌شود
→ نتایج ذخیره می‌شوند
→ نمودار به‌روزرسانی می‌شود
```

## ویژگی‌های تعاملی

### بازخورد بصری
```
تایپ صحیح:   کادر تایپ → 💚 سبز می‌شود (300ms)
تایپ اشتباه:  کادر تایپ → 🔴 قرمز و می‌لرزد (300ms)
```

### میانبرهای کیبورد
```
ESC → بستن پنجره تاریخچه
```

### ذخیره خودکار
```
هر تمرین تکمیل شده
→ ذخیره در LocalStorage
→ نمایش در نمودار
→ افزودن به تاریخچه
```

## نکات استفاده

✅ **انجام دهید:**
- تمرکز روی دقت قبل از سرعت
- تمرین منظم روزانه
- استفاده از متن‌های مختلف
- مشاهده نمودار پیشرفت

❌ **انجام ندهید:**
- نگاه کردن به کیبورد
- عجله برای افزایش سرعت
- نادیده گرفتن اشتباهات
- تمرین بدون استراحت

## مثال تمرین موفق

```
شروع: WPM = 0, دقت = 100%, زمان = 0:00
↓
تایپ کاراکتر صحیح
→ کادر سبز می‌شود
→ WPM افزایش می‌یابد
↓
تایپ کاراکتر اشتباه
→ کادر قرمز و لرزش
→ دقت کاهش می‌یابد
↓
ادامه تایپ با دقت
→ بهبود تدریجی آمار
↓
اتمام متن
→ ذخیره نتایج: WPM = 45, دقت = 95%, زمان = 1:23
→ نمایش در نمودار پیشرفت
```

---

**نکته**: این یک مثال متنی از رابط کاربری است. برای مشاهده واقعی، فایل index.html را در مرورگر باز کنید.
