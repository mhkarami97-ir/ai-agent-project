# 🚀 شروع سریع

## مشاهده محلی (بدون نیاز به چیزی)

1. فایل `index.html` را باز کنید
2. شروع به تایپ کنید!

## استقرار در GitHub Pages

### مرحله 1: ایجاد مخزن
1. به https://github.com بروید
2. روی `New repository` کلیک کنید
3. نامی مانند `typing-practice` انتخاب کنید
4. `Create repository` را بزنید

### مرحله 2: آپلود فایل‌ها
```bash
cd D:\Local\my\ai-agent-project\typing
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/typing-practice.git
git push -u origin main
```

### مرحله 3: فعال‌سازی Pages
1. Settings → Pages
2. Source: `main` branch, `/ (root)` folder
3. Save
4. منتظر بمانید (1-5 دقیقه)
5. سایت شما آماده است! 🎉
