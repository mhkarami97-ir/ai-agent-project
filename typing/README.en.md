# Typing Practice App

A beautiful and responsive typing practice web application for Persian and English languages.

🔗 **[Live Demo](https://your-username.github.io/typing/)** ← Replace with your actual GitHub Pages URL

## Quick Start

1. Fork this repository
2. Go to Settings → Pages
3. Select branch `main` and folder `/root`
4. Visit your GitHub Pages URL

## Features

- ✨ Beautiful modern UI with gradients
- 🌐 Bilingual support (Persian & English)
- 📊 WPM calculation
- 🎯 Accuracy tracking
- ⏱️ Timer
- 📈 Progress chart
- 📝 Multiple texts
- 💾 History tracking
- 📱 Fully responsive
