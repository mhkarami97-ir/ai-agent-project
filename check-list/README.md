# Checklist App

این پروژه یک اپلیکیشن ساده چک لیست با پشتیبانی از دسته بندی است که به صورت کامل در مرورگر اجرا می‌شود. داده‌ها به شکل localStorage ذخیره می‌شوند و می‌توان آن‌ها را صادر یا دوباره بارگذاری کرد.

## ساختار پروژه

- `index.html` – ورودی اصلی که ساختار صفحه، فرم ها و بخش‌های UI را تعریف می‌کند.
- `assets/css/styles.css` – استایل‌های واکنش‌گرا، حالت موبایل و اجزای فرم.
- `assets/js/app.js` – منطق دسته‌ها، آیتم‌ها، ذخیره‌سازی، وارد/صادر کردن و نمایش پیغام‌های کوتاه.

## شروع به کار

1. از مرورگر دلخواه، پوشه پروژه را باز کنید؛ کافی است فایل `index.html` را در مرورگر بکشید یا مسیر کامل را وارد کنید.
2. برای میزبانی روی GitHub Pages، پوشه را در یک ریپوی GitHub قرار دهید و تنظیمات Pages را روی شاخه‌ی حاوی `index.html` فعال کنید.

## نحوه استفاده

1. با فرم بالای بخش دسته‌بندی، یک دسته جدید ثبت و آن را فعال کنید.
2. برای هر دسته، آیتم‌های جدید را بنویسید و ذخیره کنید؛ وضعیت تکمیل‌شده روی هر آیتم نمایش داده می‌شود و از دکمه‌های کنار آن برای تغییر حالت، ویرایش یا حذف استفاده کنید.
3. دکمه‌های بخش `ریست دسته` و `نمایش خروجی` در بالای لیست آیتم‌ها امکان پاک کردن سریع یا تولید خروجی را فراهم می‌کنند.
4. بخش وارد/صادر برای کپی JSON لیست‌ها یا جایگزینی مجموعه‌ی فعلی طراحی شده است.

## بهبودها و اعتبارسنجی

- هنگام ایجاد دسته جدید، عنوان نمی‌تواند خالی باشد و برای نمایش در لیست، محدودیت مرورگر روی فرم اعمال شده است.
- بعد از هر تغییر، وضعیت در localStorage ذخیره شده و با بارگذاری مجدد از سر گرفته می‌شود.
- واردسازی، فقط در صورت ساختار لیست صحیح، جایگزین خواهد شد و در غیر این صورت پیغام خطا نمایش داده می‌شود.

## تست دستی پیشنهادی

1. ثبت دسته، افزودن آیتم، تغییر وضعیت، و تایید باقی ماندن بعد از رفرش.
2. حذف و ویرایش آیتم، سپس ریست دسته برای خالی کردن لیست.
3. خروجی گرفتن و وارد کردن داده‌ها با JSON معتبر/نامعتبر برای بررسی پیام‌ها.
4. باز کردن صفحه در ابعاد موبایل برای تایید چینش واکنش‌گرا.
