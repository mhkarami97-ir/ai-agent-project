# برنامه مدیریت دارو

این پروژه یک اپ سادهٔ مدیریت دارو برای استفادهٔ شخصی است که به صورت صفحات HTML/CSS/JS نوشته شده و می‌توان آن را روی GitHub Pages میزبانی کرد.

## ساختار فایل
- `index.html` — صفحهٔ اصلی و رابط کاربری
- `assets/styles.css` — تمام استایل‌ها
- `assets/scripts.js` — منطق ثبت، هشدار و وضعیت دارو

## ویژگی‌ها
1. رابط کاربری راست‌چین با فونت Vazirmatn
2. ثبت دارو با دوز، زمان، یادداشت و حالت «خوردم/نخوردم»
3. نمایش تعداد کل داروها و آن‌هایی که در ۳۰ دقیقهٔ آینده موعد دارند
4. نمایش هشدار نزدیک‌ترین داروها به همراه وضعیت مصرف
5. ذخیره‌سازی تمام داده‌ها در حافظهٔ مرورگر (`localStorage`)
6. طراحی واکنش‌گرا برای نمایش موبایلی

## نحوه استفاده
1. پوشهٔ پروژه را کلون کنید یا فایل‌ها را دانلود کنید
2. پوشه را روی GitHub Pages قرار دهید یا به سادگی `index.html` را در مرورگر باز کنید
3. فرم بالا را پر کنید تا داروها ذخیره شوند و در لیست نمایش داده شوند
4. روی دکمه‌های «خوردم» یا «نخوردم» کلیک کنید تا وضعیت ثبت شود

## توسعه
- فایل‌های CSS و JS در مسیر `assets/` جدا نگه داشته شدند
- برای اضافه کردن قابلیت جدید می‌توانید به `assets/scripts.js` رجوع کنید

## تست و تجربه
- مرورگر را باز کرده و `index.html` را اجرا کنید
- فرم را پر کرده و وضعیت داروها را مشاهده کنید
- به بخش اعلان‌ها نگاه کنید تا نزدیکترین مصرف‌ها را ببینید
