# مقایسه‌گر متن (Text Diff Comparator)

یک ابزار حرفه‌ای و کاربردی برای مقایسه دو متن و نمایش تفاوت‌های آنها به صورت بصری و در لحظه، مشابه ابزارهای مقایسه کد در Git.

## 🌟 ویژگی‌های کلیدی

### قابلیت‌های اصلی
- **مقایسه لحظه‌ای**: تغییرات به صورت خودکار و در لحظه نمایش داده می‌شوند
- **هایلایت رنگی**: نمایش تغییرات با رنگ‌های متمایز:
  - 🟢 **سبز**: خطوط افزوده شده
  - 🔴 **قرمز**: خطوط حذف شده
  - 🟡 **زرد**: خطوط تغییر یافته
  - ⚪ **خاکستری**: خطوط بدون تغییر
- **نمایش side-by-side**: مشاهده همزمان متن اصلی و جدید در کنار هم
- **آمار دقیق**: نمایش تعداد خطوط افزوده شده، حذف شده و بدون تغییر
- **ذخیره خودکار**: متن‌های وارد شده به صورت خودکار در مرورگر ذخیره می‌شوند

### امکانات کاربردی
- **جابه‌جایی متون**: تعویض سریع جایگاه دو متن
- **پاک کردن سریع**: حذف تمام محتوا با یک کلیک
- **شمارش خطوط**: نمایش تعداد خطوط هر متن
- **طراحی رسپانسیو**: سازگار با تمام دستگاه‌ها (دسکتاپ، تبلت، موبایل)
- **رابط کاربری RTL**: طراحی کامل برای زبان فارسی

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
این پروژه نیازی به نصب وابستگی ندارد و به صورت کامل در مرورگر اجرا می‌شود.

### نحوه استفاده محلی

1. **دانلود پروژه**
   ```bash
   git clone https://github.com/YOUR-USERNAME/compare.git
   cd compare
   ```

2. **اجرای پروژه**
   - فایل `index.html` را در مرورگر خود باز کنید
   - یا از یک وب‌سرور محلی استفاده کنید:
   ```bash
   # با Python 3
   python -m http.server 8000
   
   # با Node.js (npx)
   npx serve
   ```

3. **دسترسی به برنامه**
   - در مرورگر به آدرس `http://localhost:8000` بروید

### استقرار در GitHub Pages

1. **آپلود کد در GitHub**
   - پروژه را در مخزن GitHub خود قرار دهید
   - به Settings > Pages بروید

2. **فعال‌سازی GitHub Pages**
   - Source را روی `main` یا `master` برنچ تنظیم کنید
   - پوشه root را انتخاب کنید
   - Save را بزنید

3. **دسترسی به سایت**
   - پس از چند دقیقه، سایت شما در آدرس زیر در دسترس خواهد بود:
   - `https://YOUR-USERNAME.github.io/compare/`

## 🎯 نحوه استفاده

### استفاده پایه
1. **ورود متن اصلی**: متن اولیه خود را در کادر سمت راست وارد کنید
2. **ورود متن جدید**: متن تغییر یافته را در کادر سمت چپ وارد کنید
3. **مشاهده نتایج**: تغییرات به صورت خودکار نمایش داده می‌شوند
4. **بررسی آمار**: آمار کامل تغییرات در پایین صفحه قابل مشاهده است

### دکمه‌های کنترلی
- **مقایسه**: انجام مقایسه دستی (در صورت نیاز)
- **جابه‌جایی**: تعویض جایگاه دو متن
- **پاک کردن**: حذف تمام محتوای وارد شده

### نکات کاربردی
- متن‌های شما به صورت خودکار در مرورگر ذخیره می‌شوند
- می‌توانید صفحه را ببندید و بعداً بازگردید - متن‌ها حفظ خواهند شد
- برای مقایسه کدها، متن‌های چند خطی، مقالات و ... مناسب است
- در موبایل، متن‌ها به صورت عمودی نمایش داده می‌شوند

## 🛠️ جزئیات فنی

### الگوریتم مقایسه
- استفاده از الگوریتم **LCS (Longest Common Subsequence)**
- پیاده‌سازی با **برنامه‌نویسی پویا (Dynamic Programming)**
- پیچیدگی زمانی: O(m × n) که m و n طول دو متن هستند
- دقت بالا در تشخیص تغییرات

### ذخیره‌سازی داده
- استفاده از **localStorage** مرورگر
- بدون نیاز به سرور یا دیتابیس
- ذخیره خودکار پس از هر تغییر
- حفظ اطلاعات حتی پس از بستن مرورگر

### سازگاری مرورگرها
- ✅ Chrome/Edge (نسخه‌های جدید)
- ✅ Firefox (نسخه‌های جدید)
- ✅ Safari (نسخه‌های جدید)
- ✅ Opera (نسخه‌های جدید)

### پشتیبانی از دستگاه‌ها
- 💻 **دسکتاپ**: نمایش side-by-side
- 📱 **تبلت**: نمایش side-by-side با فضای کمتر
- 📱 **موبایل**: نمایش عمودی (stacked)

## 🎨 سفارشی‌سازی

### تغییر رنگ‌ها
برای تغییر رنگ‌های پوسته، متغیرهای CSS در فایل `assets/style.css` را ویرایش کنید:

```css
:root {
    --primary-color: #4f46e5;      /* رنگ اصلی */
    --added-bg: #dcfce7;           /* پس‌زمینه افزوده شده */
    --removed-bg: #fee2e2;         /* پس‌زمینه حذف شده */
    --modified-bg: #fef3c7;        /* پس‌زمینه تغییر یافته */
}
```

### تغییر فونت
فونت Vazirmatn از Google Fonts بارگذاری می‌شود. برای تغییر فونت، خط زیر را در `index.html` ویرایش کنید:

```html
<link href="https://fonts.googleapis.com/css2?family=YOUR-FONT&display=swap" rel="stylesheet">
```
