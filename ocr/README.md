# 🔍 تشخیص متن از تصویر (OCR Web Application)

یک برنامه تحت وب پیشرفته برای تشخیص و استخراج متن از تصاویر با پشتیبانی از زبان‌های مختلف و قابلیت کار آفلاین.

## ✨ ویژگی‌ها

### 🎯 ویژگی‌های اصلی
- **تشخیص متن هوشمند**: استخراج دقیق متن از انواع تصاویر
- **پشتیبانی چندزبانه**: پشتیبانی از ۱۰ زبان مختلف شامل فارسی، انگلیسی، عربی و...
- **حالت آفلاین**: قابلیت کار در صورت عدم دسترسی به اینترنت (حالت نمایشی)
- **مکانیزم بازیابی هوشمند**: تلاش خودکار با CDN‌های مختلف در صورت خطا
- **رابط کاربری RTL**: طراحی مناسب برای زبان‌های راست به چپ
- **فونت Vazirmatn**: استفاده از فونت زیبای وزیر متن
- **طراحی ریسپانسیو**: سازگار با تمامی اندازه‌های صفحه نمایش

### 🛠 ویژگی‌های فنی
- **بدون نیاز به سرور**: تمام پردازش در مرورگر انجام می‌شود
- **ذخیره‌سازی محلی**: تاریخچه در دیتابیس مرورگر ذخیره می‌شود
- **عملکرد بالا**: بهینه‌سازی شده برای سرعت و کارایی
- **امنیت**: هیچ اطلاعاتی به سرور ارسال نمی‌شود
- **مقاوم در برابر خطا**: مدیریت هوشمند خطاها و ارائه گزینه‌های جایگزین

### 🎨 ویژگی‌های کاربری
- **کشیدن و رها کردن**: آپلود آسان فایل با Drag & Drop
- **پیش‌نمایش زنده**: نمایش تصویر قبل از پردازش
- **نوار پیشرفت پیشرفته**: نمایش دقیق وضعیت پردازش
- **نشانگر وضعیت شبکه**: نمایش وضعیت اتصال اینترنت
- **کپی سریع**: کپی متن با یک کلیک
- **دانلود متن**: ذخیره متن به صورت فایل
- **تاریخچه کامل**: مدیریت تاریخچه تشخیص‌ها

## 🚀 نحوه استفاده

### 1️⃣ انتخاب تصویر
- روی منطقه آپلود کلیک کنید یا فایل را بکشید و رها کنید
- تصاویر با فرمت‌های JPG، PNG، GIF، BMP پشتیبانی می‌شوند
- حداکثر اندازه فایل: ۱۰ مگابایت

### 2️⃣ انتخاب زبان
- زبان متن موجود در تصویر را انتخاب کنید
- زبان‌های پشتیبانی شده:
  - 🇮🇷 فارسی
  - 🇺🇸 انگلیسی  
  - 🇸🇦 عربی
  - 🇫🇷 فرانسوی
  - 🇩🇪 آلمانی
  - 🇪🇸 اسپانیولی
  - 🇹🇷 ترکی
  - 🇷🇺 روسی
  - 🇨🇳 چینی ساده
  - 🇯🇵 ژاپنی

### 3️⃣ پردازش تصویر
- دکمه "شروع تشخیص متن" را کلیک کنید
- منتظر تکمیل پردازش بمانید (معمولاً ۵-۳۰ ثانیه)
- متن استخراج شده نمایش داده می‌شود

### 4️⃣ مدیریت نتایج
- **کپی متن**: متن را در کلیپ‌بورد کپی کنید
- **دانلود متن**: متن را به صورت فایل .txt دانلود کنید
- **ذخیره در تاریخچه**: نتیجه را در تاریخچه ذخیره کنید

## 💻 فناوری‌های استفاده شده

### Frontend
- **HTML5**: ساختار صفحه
- **CSS3**: طراحی و استایل‌دهی
- **JavaScript (ES6+)**: منطق برنامه
- **Tesseract.js**: موتور تشخیص متن

### طراحی و UX/UI
- **Vazirmatn Font**: فونت اصلی برنامه
- **Responsive Design**: سازگاری با دستگاه‌های مختلف
- **RTL Layout**: پشتیبانی از زبان‌های راست به چپ
- **Modern UI**: رابط کاربری مدرن و زیبا

### ذخیره‌سازی
- **LocalStorage API**: ذخیره تاریخچه در مرورگر
- **IndexedDB**: پشتیبان برای ذخیره‌سازی پیشرفته

## 🛠 نصب و راه‌اندازی

### روش اول: استفاده مستقیم
1. فایل‌ها را دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید
3. برنامه آماده استفاده است!

### روش دوم: سرور محلی
```bash
# استفاده از Python
python -m http.server 8000

# یا استفاده از Node.js
npx http-server

# یا استفاده از PHP
php -S localhost:8000
```

### روش سوم: GitHub Pages
1. مخزن را Fork کنید
2. در تنظیمات مخزن، GitHub Pages را فعال کنید
3. از طریق URL ارائه شده به برنامه دسترسی پیدا کنید

## 📁 ساختار پروژه

```
ocr/
├── 📄 index.html          # صفحه اصلی برنامه
├── 📁 assets/             # فایل‌های استاتیک
│   ├── 🎨 style.css       # فایل استایل اصلی
│   └── ⚙️ script.js       # فایل جاوا اسکریپت اصلی
└── 📖 README.md           # مستندات پروژه
```

### توضیح فایل‌ها

#### `index.html`
- ساختار HTML صفحه اصلی
- لینک به فایل‌های CSS و JavaScript
- تعریف عناصر رابط کاربری

#### `assets/style.css`
- طراحی ریسپانسیو و مدرن
- پشتیبانی از RTL
- استایل‌های مخصوص فونت Vazirmatn
- انیمیشن‌ها و جلوه‌های بصری

#### `assets/script.js`
- کلاس اصلی `OCRApp`
- مدیریت آپلود فایل
- پردازش OCR با Tesseract.js
- مدیریت تاریخچه و localStorage
- مکانیزم بازیابی خطا و فالبک

## 🔧 حل مشکلات

### مشکلات رایج و راه‌حل‌ها

#### ❌ خطای "Failed to fetch"
**علت**: عدم دسترسی به CDN یا مشکل اتصال اینترنت

**راه‌حل‌ها**:
1. **بررسی اتصال اینترنت**: اطمینان حاصل کنید که اتصال اینترنت برقرار است
2. **استفاده از حالت نمایشی**: برنامه به صورت خودکار وارد حالت نمایشی می‌شود
3. **تلاش مجدد**: بعد از برقراری اتصال، دوباره تلاش کنید
4. **استفاده از VPN**: در صورت فیلتر بودن CDN، از VPN استفاده کنید

#### ⚠️ حالت نمایشی (Demo Mode)
**زمان فعال‌سازی**: 
- عدم دسترسی به Tesseract.js
- مشکل در بارگذاری فایل‌های زبان
- خطای شبکه

**محدودیت‌ها**:
- تشخیص واقعی متن انجام نمی‌شود
- متن نمایشی ایجاد می‌شود
- تمام ویژگی‌های دیگر کار می‌کنند

#### 🌐 مشکلات مرورگر
**مشکل**: عدم پشتیبانی از برخی ویژگی‌ها

**راه‌حل**:
- از آخرین نسخه مرورگر استفاده کنید
- JavaScript را فعال کنید
- LocalStorage را فعال کنید

#### 📱 مشکلات موبایل
**مشکل**: کندی در پردازش

**راه‌حل**:
- از تصاویر با رزولوشن کمتر استفاده کنید
- از زبان انگلیسی برای سرعت بیشتر استفاده کنید
- صفحه را بازخوانی کنید

### نکات بهینه‌سازی

#### 🖼️ کیفیت تصویر
- **رزولوشن**: حداقل 300 DPI برای نتایج بهتر
- **کنتراست**: تصاویر با کنتراست بالا
- **نور**: تصاویر با نور مناسب و بدون سایه
- **زاویه**: تصاویر مستقیم و بدون انحنا

#### 🔤 نوع متن
- **فونت واضح**: فونت‌های خوانا مثل Arial یا Times
- **اندازه مناسب**: متن نه خیلی کوچک و نه خیلی بزرگ
- **رنگ**: متن مشکی روی پس‌زمینه سفید بهترین است

#### 🌐 عملکرد شبکه
- **اتصال پایدار**: اتصال اینترنت پایدار برای اولین استفاده
- **صبر کنید**: فرآیند اولیه ممکن است کند باشد
- **کش مرورگر**: فایل‌ها در کش ذخیره می‌شوند
