## 💡 How to Use | نحوه استفاده

<div dir="rtl">

### فارسی:
1. **باز کردن**: index.html را باز کنید
2. **تغییر زبان**: روی دکمه زبان کلیک کنید
3. **اسکن**: دستگاه‌های بلوتوث را اسکن کنید
4. **تنظیم**: پارامترهای حمله را تنظیم کنید
5. **شروع**: حمله را شروع کنید
6. **مشاهده**: آمار و لاگ‌ها را ببینید

</div>

### English:
1. **Open**: Open index.html
2. **Switch**: Click language toggle button
3. **Scan**: Scan for Bluetooth devices
4. **Configure**: Set attack parameters
5. **Start**: Begin the attack
6. **Monitor**: Watch stats and logs
