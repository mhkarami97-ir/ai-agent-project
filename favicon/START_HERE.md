### استقرار در GitHub Pages

#### مرحله 1: ایجاد مخزن
1. به [GitHub.com](https://github.com) بروید
2. New repository را بزنید
3. نامی مانند `icon-generator` بدهید
4. Public را انتخاب کنید
5. Create repository را بزنید

#### مرحله 2: آپلود فایل‌ها
دو روش دارید:

**روش ساده (از طریق وب):**
1. در صفحه مخزن: "uploading an existing file"
2. همه فایل‌ها را بکشید و رها کنید (به جز پوشه .idea)
3. Commit changes

**روش Git:**
```bash
cd "D:\Local\my\ai-agent-project\favicon"
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```

#### مرحله 3: فعال‌سازی Pages
1. Settings > Pages
2. Branch: main
3. Folder: / (root)
4. Save

✅ سایت شما در چند دقیقه آماده است:
```
https://USERNAME.github.io/REPO/
```

## 🎨 نحوه کار با برنامه

### 1. آپلود تصویر
- کلیک روی "انتخاب تصویر" یا
- کشیدن و رها کردن تصویر

**توصیه**: تصویر مربعی با کیفیت بالا (حداقل 512×512)

### 2. تنظیمات
- **رنگ پس‌زمینه**: شفاف یا رنگی
- **فاصله حاشیه**: 0% تا 30% (پیشنهاد: 10%)

### 3. تولید
کلیک روی "تولید آیکون‌ها"

### 4. دانلود
- تک‌تک: دکمه دانلود زیر هر آیکون
- همه: دکمه "دانلود همه" (فایل ZIP)

### 5. تاریخچه
آیکون‌های قبلی در تاریخچه نگهداری می‌شوند.

## 📋 سایزهای تولیدی

### Favicon ICO
- **favicon.ico** - فایل ICO شامل 16×16, 32×32, 48×48 (سازگار با همه مرورگرها)

### Favicon PNG (برای وب‌سایت)
- 16×16, 32×32, 48×48, 64×64

### PWA Icons (برای اپلیکیشن)
- 128×128, 144×144, 152×152, 167×167, 180×180
- 192×192, 256×256, 384×384, 512×512

## 💡 نکات مهم

### استفاده از آیکون‌ها

در `<head>` صفحه HTML:
```html
<link rel="icon" sizes="16x16" href="/favicons/favicon-16x16.png">
<link rel="icon" sizes="32x32" href="/favicons/favicon-32x32.png">
```

برای PWA در `manifest.json`:
```json
{
  "icons": [
    {"src": "/pwa-icons/icon-192x192.png", "sizes": "192x192"},
    {"src": "/pwa-icons/icon-512x512.png", "sizes": "512x512"}
  ]
}
```
