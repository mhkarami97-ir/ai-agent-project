# 🎯 فرمت ICO اضافه شد!

## ✨ ویژگی جدید

برنامه حالا **فایل ICO** نیز تولید می‌کند که سازگار با تمام مرورگرها است.

## 📦 خروجی‌ها

### 1. فایل ICO
- **نام فایل**: `favicon.ico`
- **سایزها**: شامل 16×16, 32×32, 48×48
- **سازگاری**: تمام مرورگرها (حتی نسخه‌های قدیمی IE)
- **نمایش**: با آیکون مخصوص در نتایج

### 2. فایل‌های PNG
- 13 سایز مختلف (16×16 تا 512×512)
- برای Favicon و PWA

## 🎨 نحوه استفاده

### استفاده از فایل ICO (پیشنهادی)

فایل `favicon.ico` را در روت سایت خود قرار دهید:

```html
<link rel="icon" href="/favicon.ico" type="image/x-icon">
<link rel="shortcut icon" href="/favicon.ico" type="image/x-icon">
```

### استفاده از PNG (مدرن)

برای مرورگرهای جدید:

```html
<link rel="icon" type="image/png" sizes="16x16" href="/favicons/favicon-16x16.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicons/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="48x48" href="/favicons/favicon-48x48.png">
```

### استفاده از هر دو (بهترین)

برای سازگاری کامل:

```html
<!-- فایل ICO برای سازگاری -->
<link rel="icon" href="/favicon.ico" type="image/x-icon">

<!-- فایل‌های PNG برای مرورگرهای جدید -->
<link rel="icon" type="image/png" sizes="16x16" href="/favicons/favicon-16x16.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicons/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="48x48" href="/favicons/favicon-48x48.png">
```

## 📥 دانلود

### دانلود تکی
روی دکمه "دانلود" زیر فایل `favicon.ico` کلیک کنید.

### دانلود با ZIP
فایل ICO به طور خودکار در ZIP گنجانده می‌شود:
```
icons-TIMESTAMP.zip
├── favicon.ico          ← فایل ICO
├── favicons/            ← PNG Favicons
├── pwa-icons/           ← PWA Icons
└── README.md            ← راهنما
```
