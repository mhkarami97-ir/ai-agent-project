# 📋 راهنمای استفاده

## 🎯 درباره برنامه

این برنامه یک ابزار وب برای تولید آیکون‌های Favicon و PWA است که به شما امکان می‌دهد با آپلود یک تصویر، به راحتی تمام سایزهای مورد نیاز را تولید و دانلود کنید.

## 🚀 شروع سریع

### 1. آپلود تصویر

دو روش برای آپلود تصویر وجود دارد:

- **روش اول**: روی دکمه "انتخاب تصویر" کلیک کرده و فایل خود را انتخاب کنید
- **روش دوم**: تصویر را بکشید و روی ناحیه آپلود رها کنید (Drag & Drop)

**توصیه**: از تصاویر با کیفیت بالا و نسبت ابعاد مربعی (1:1) استفاده کنید.

### 2. تنظیمات

پس از آپلود تصویر، می‌توانید تنظیمات زیر را انجام دهید:

#### رنگ پس‌زمینه
- **شفاف**: برای پس‌زمینه شفاف این گزینه را فعال کنید (پیشنهاد می‌شود)
- **رنگی**: می‌توانید رنگ دلخواه خود را انتخاب کنید یا کد رنگ را وارد کنید

#### فاصله از حاشیه
- با استفاده از اسلایدر، می‌توانید فاصله تصویر از حاشیه را تنظیم کنید
- محدوده: 0% تا 30%
- پیشنهاد: 10% برای لوگوها و 5% برای آیکون‌های پر

### 3. تولید آیکون‌ها

روی دکمه "تولید آیکون‌ها" کلیک کنید. برنامه به طور خودکار موارد زیر را تولید می‌کند:

#### Favicon ICO
- **favicon.ico** - فایل ICO شامل سایزهای 16x16, 32x32, 48x48 (سازگار با تمام مرورگرها)

#### Favicon PNG Sizes
- 16×16 پیکسل
- 32×32 پیکسل
- 48×48 پیکسل
- 64×64 پیکسل

#### PWA Icon Sizes
- 128×128 پیکسل
- 144×144 پیکسل
- 152×152 پیکسل
- 167×167 پیکسل
- 180×180 پیکسل
- 192×192 پیکسل
- 256×256 پیکسل
- 384×384 پیکسل
- 512×512 پیکسل

### 4. دانلود

دو گزینه برای دانلود دارید:

- **دانلود تک‌تک**: روی دکمه "دانلود" زیر هر آیکون کلیک کنید
- **دانلود همه**: روی دکمه "دانلود همه" کلیک کنید تا تمام آیکون‌ها در یک فایل ZIP دانلود شوند

فایل ZIP شامل:
- پوشه `favicons` با آیکون‌های Favicon
- پوشه `pwa-icons` با آیکون‌های PWA
- فایل `README.md` با راهنمای استفاده

## 💾 تاریخچه

برنامه به طور خودکار آیکون‌های تولید شده را در تاریخچه ذخیره می‌کند:

- آیکون‌های قبلی خود را مشاهده کنید
- با کلیک روی هر آیتم، دوباره از آن استفاده کنید
- برای پاک کردن تاریخچه، روی "پاک کردن تاریخچه" کلیک کنید

**نکته**: تاریخچه در مرورگر شما ذخیره می‌شود و هیچ اطلاعاتی به سرور ارسال نمی‌شود.

## 🎨 نکات و ترفندها

### بهترین تصاویر

1. **فرمت**: PNG یا SVG با پس‌زمینه شفاف
2. **سایز**: حداقل 512×512 پیکسل
3. **نسبت**: مربع (1:1)
4. **محتوا**: ساده و واضح، قابل تشخیص در سایزهای کوچک

### تنظیمات پیشنهادی

#### برای لوگوها با متن:
- فاصله حاشیه: 5-8%
- پس‌زمینه: شفاف یا رنگ ساده

#### برای آیکون‌های گرافیکی:
- فاصله حاشیه: 10-15%
- پس‌زمینه: شفاف

#### برای عکس‌های شخصی:
- فاصله حاشیه: 0-5%
- پس‌زمینه: رنگی یا شفاف

## 🔧 استفاده در پروژه

### اضافه کردن Favicon ICO (پیشنهادی)

فایل `favicon.ico` را در روت سایت خود قرار دهید و در `<head>` بنویسید:

```html
<link rel="icon" href="/favicon.ico" type="image/x-icon">
<link rel="shortcut icon" href="/favicon.ico" type="image/x-icon">
```

**نکته**: فایل ICO با تمام مرورگرها و نسخه‌های قدیمی سازگار است.

### اضافه کردن Favicon PNG

فایل‌های تولید شده را در پوشه عمومی پروژه خود قرار دهید و در `<head>` بنویسید:

```html
<link rel="icon" type="image/png" sizes="16x16" href="/favicons/favicon-16x16.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicons/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="48x48" href="/favicons/favicon-48x48.png">
```

### استفاده در PWA

فایل `manifest.json` ایجاد کنید:

```json
{
  "name": "نام برنامه",
  "short_name": "نام کوتاه",
  "icons": [
    {
      "src": "/pwa-icons/icon-192x192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "/pwa-icons/icon-512x512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ],
  "theme_color": "#ffffff",
  "background_color": "#ffffff",
  "display": "standalone"
}
```

در `<head>` اضافه کنید:

```html
<link rel="manifest" href="/manifest.json">
<meta name="theme-color" content="#ffffff">
```

### آیا تصاویر من آپلود می‌شوند؟
خیر! تمام پردازش در مرورگر شما انجام می‌شود و هیچ فایلی به سرور ارسال نمی‌شود.

### کیفیت آیکون‌ها خوب است؟

تمام فرمت‌های تصویری که مرورگر پشتیبانی می‌کند: PNG, JPG, JPEG, GIF, SVG, WEBP

### چرا تاریخچه پاک شد؟
تاریخچه در مرورگر شما ذخیره می‌شود. اگر حافظه مرورگر را پاک کنید، تاریخچه هم پاک می‌شود.

### چطور می‌توانم از این برنامه به صورت آفلاین استفاده کنم؟
