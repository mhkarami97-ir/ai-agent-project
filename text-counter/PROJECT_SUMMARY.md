## 🚀 نحوه استفاده

### استفاده محلی
1. فایل `index.html` را در مرورگر باز کنید
2. شروع به استفاده کنید!

### استقرار در GitHub Pages
1. یک مخزن جدید در GitHub ایجاد کنید
2. فایل‌ها را آپلود کنید
3. در Settings > Pages گزینه GitHub Actions را فعال کنید
4. سایت شما آماده است!

راهنمای کامل در فایل `DEPLOYMENT.md` موجود است.
