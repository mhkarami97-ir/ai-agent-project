# 🔐 تولید کننده رمز عبور امن

یک برنامه وب آفلاین برای تولید رمزهای عبور قوی و امن با قابلیت‌های پیشرفته و رابط کاربری فارسی.

## ✨ ویژگی‌ها

### تولید رمز عبور قدرتمند
- **تولید رمز عبور امن**: استفاده از `crypto.getRandomValues` برای تولید رمزهای عبور با امنیت بالا
- **قابل تنظیم بودن طول**: انتخاب طول رمز عبور از 4 تا 64 کاراکتر
- **انواع مختلف کاراکترها**:
  - حروف بزرگ انگلیسی (A-Z)
  - حروف کوچک انگلیسی (a-z)
  - اعداد (0-9)
  - نمادها و کاراکترهای خاص (!@#$%^&*...)

### سیاست‌های امنیتی پیشرفته
- **حذف کاراکترهای مشابه**: جلوگیری از استفاده کاراکترهای گیج‌کننده مانند `i`, `l`, `1`, `L`, `o`, `0`, `O`
- **حذف نمادهای مبهم**: حذف نمادهایی که ممکن است در برخی سیستم‌ها مشکل ایجاد کنند مانند `{}[]()/\'\"~,;:.<>`

### نمایش قدرت رمز عبور
- **سنجش قدرت**: تحلیل هوشمند قدرت رمز عبور
- **نمایش بصری**: نوار پیشرفت رنگی برای نمایش قدرت (ضعیف، متوسط، خوب، عالی)
- **راهنمایی**: پیشنهادات برای بهبود امنیت رمز عبور

### تاریخچه رمزهای عبور
- **ذخیره خودکار**: ذخیره 10 رمز عبور آخر
- **کپی سریع**: کپی کردن رمزهای قبلی با یک کلیک
- **مدیریت آسان**: پاک کردن تاریخچه در صورت نیاز

### رابط کاربری
- **طراحی مدرن**: رابط کاربری زیبا با انیمیشن‌های روان
- **فارسی و RTL**: کاملاً فارسی با پشتیبانی کامل از راست‌به‌چپ
- **فونت Vazirmatn**: استفاده از فونت زیبای وزیرمتن
- **رسپانسیو**: سازگار با موبایل، تبلت و دسکتاپ
- **تم جذاب**: گرادیانت‌های رنگی و المان‌های بصری زیبا

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- یک مرورگر مدرن (Chrome, Firefox, Safari, Edge)
- هیچ نیازی به سرور یا بکند ندارد

### روش اول: استفاده مستقیم
1. فایل‌ها را دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید
3. شروع به استفاده کنید!

### روش دوم: میزبانی در GitHub Pages
1. یک مخزن جدید در GitHub ایجاد کنید
2. فایل‌های پروژه را به مخزن اضافه کنید:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPOSITORY.git
   git push -u origin main
   ```
3. به تنظیمات مخزن بروید (Settings)
4. در بخش Pages، منبع را روی `main` برنچ تنظیم کنید
5. چند دقیقه صبر کنید تا سایت منتشر شود
6. سایت شما در آدرس `https://USERNAME.github.io/REPOSITORY` در دسترس خواهد بود

## 💡 نحوه استفاده

### تولید رمز عبور
1. طول دلخواه را با اسلایدر انتخاب کنید
2. نوع کاراکترهای مورد نظر را انتخاب کنید (حروف بزرگ، کوچک، اعداد، نمادها)
3. در صورت نیاز، گزینه‌های امنیتی اضافی را فعال کنید
4. روی دکمه "تولید رمز عبور" کلیک کنید
5. رمز عبور تولید شده را مشاهده و کپی کنید

### تنظیمات پیشرفته
- **حذف کاراکترهای مشابه**: برای جلوگیری از اشتباه در تایپ دستی
- **حذف نمادهای مبهم**: برای سازگاری بیشتر با سیستم‌های مختلف

### مشاهده تاریخچه
- آخرین 10 رمز عبور تولید شده در بخش تاریخچه نمایش داده می‌شود
- می‌توانید هر کدام را مجدداً کپی کنید
- با دکمه "پاک کردن تاریخچه" می‌توانید تاریخچه را حذف کنید

## 🔒 امنیت و حریم خصوصی

### ذخیره‌سازی محلی
- تمام داده‌ها در مرورگر شما ذخیره می‌شوند (localStorage)
- هیچ اطلاعاتی به سرور ارسال نمی‌شود
- کاملاً آفلاین کار می‌کند

### تولید امن
- استفاده از Web Crypto API (`crypto.getRandomValues`)
- تصادفی‌سازی با کیفیت رمزنگاری
- عدم وابستگی به `Math.random()` که امنیت کمتری دارد

### توصیه‌های امنیتی
- ✅ رمزهای عبور با طول حداقل 12 کاراکتر
- ✅ استفاده از ترکیبی از حروف، اعداد و نمادها
- ✅ عدم استفاده مجدد از یک رمز عبور برای چند حساب
- ✅ تغییر دوره‌ای رمزهای عبور مهم

- تعداد رمزهای ذخیره شده در تاریخچه
- مجموعه کاراکترهای استفاده شده
