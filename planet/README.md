# 🌱 گیاه من - Plant Manager

برنامه مدیریت هوشمند گیاهان با قابلیت یادآوری و پیگیری نگهداری

## 📋 ویژگی‌ها

### ✨ امکانات اصلی

- **مدیریت گیاهان**: افزودن، ویرایش و حذف گیاهان با مشخصات کامل
- **آپلود تصویر**: امکان افزودن عکس برای هر گیاه
- **یادآوری هوشمند**: یادآوری خودکار برای آبیاری و کوددهی
- **دسته‌بندی**: سازماندهی گیاهان با تگ‌ها و مکان قرارگیری
- **جستجو و فیلتر**: جستجوی سریع و فیلتر بر اساس نیاز آبی، نوری و مکان
- **راهنمای کامل**: بخش راهنمای جامع برای نگهداری از گیاهان
- **آمار و گزارش**: نمایش آمار و نمودارهای مختلف
- **پشتیبان‌گیری**: امکان Import/Export اطلاعات به صورت JSON

### 🎯 قابلیت‌های پیشرفته

- **ذخیره‌سازی محلی**: تمام اطلاعات در مرورگر (LocalStorage) ذخیره می‌شود
- **بدون نیاز به سرور**: کاملاً مستقل و بدون نیاز به بکند
- **رابط کاربری فارسی**: کاملاً به زبان فارسی با پشتیبانی RTL
- **ریسپانسیو**: سازگار با موبایل، تبلت و دسکتاپ
- **طراحی مدرن**: ظاهر زیبا و کاربرپسند

## 🚀 نحوه استفاده

### نصب و راه‌اندازی

1. فایل‌های فونت Vazirmatn را در پوشه `assets/fonts/` قرار دهید:
   ```
   assets/
   ├── fonts/
   │   ├── Vazirmatn-Regular.woff2
   │   ├── Vazirmatn-Regular.woff
   │   ├── Vazirmatn-Medium.woff2
   │   ├── Vazirmatn-Medium.woff
   │   ├── Vazirmatn-Bold.woff2
   │   ├── Vazirmatn-Bold.woff
   │   ├── Vazirmatn-Light.woff2
   │   └── Vazirmatn-Light.woff
   ```

2. فایل `index.html` را در مرورگر باز کنید

3. شروع به استفاده کنید!

### افزودن گیاه جدید

1. روی دکمه "➕ افزودن گیاه" کلیک کنید
2. اطلاعات گیاه را وارد کنید:
   - نام گیاه (الزامی)
   - نام علمی (اختیاری)
   - نیاز آبی: کم / متوسط / زیاد
   - نیاز نوری: کم / متوسط / زیاد
   - بازه آبیاری (به روز)
   - بازه کوددهی (به روز - اختیاری)
   - مکان قرارگیری
   - برچسب‌ها
   - یادداشت
   - تصویر گیاه
3. روی "ذخیره گیاه" کلیک کنید

### مدیریت یادآوری‌ها

- یادآوری‌های سررسید شده با رنگ قرمز نمایش داده می‌شوند
- با کلیک روی دکمه آبیاری/کوددهی، وظیفه انجام می‌شود
- در بخش "یادآوری‌ها" می‌توانید لیست کامل وظایف را مشاهده کنید
- با تیک زدن هر یادآوری، وظیفه به عنوان انجام شده ثبت می‌شود

### فیلتر و جستجو

- **جستجو**: در کادر جستجو می‌توانید نام گیاه یا نام علمی را جستجو کنید
- **فیلتر**: روی برچسب‌های بالای لیست می‌توانید گیاهان را بر اساس:
  - مکان قرارگیری
  - نیاز آبی (کم/متوسط/زیاد)
  - نیاز نوری (کم/متوسط/زیاد)
  - تگ‌های سفارشی

### پشتیبان‌گیری و بازیابی

#### خروجی گرفتن (Export):
1. روی دکمه "📤" در هدر کلیک کنید
2. فایل JSON با نام `plant-data-YYYY-MM-DD.json` دانلود می‌شود
3. این فایل را در جای امن نگهداری کنید

#### ورود اطلاعات (Import):
1. روی دکمه "📥" در هدر کلیک کنید
2. فایل JSON قبلی را انتخاب کنید
3. اطلاعات بازیابی می‌شود

## 📖 راهنمای نگهداری گیاهان

برنامه شامل یک بخش جامع راهنمای نگهداری است که شامل:

- آبیاری صحیح (کم، متوسط، زیاد)
- نور مناسب و نحوه قرارگیری گیاه
- دما و رطوبت مناسب
- کوددهی اصولی
- هرس و نگهداری
- مقابله با آفات و بیماری‌ها
- معرفی گیاهان مناسب برای مبتدی‌ها

## 📊 آمار و گزارش

در بخش آمار می‌توانید موارد زیر را مشاهده کنید:

- تعداد کل گیاهان
- تعداد وظایف امروز
- تعداد وظایف انجام شده این ماه
- تعداد مکان‌های مختلف
- نمودار توزیع گیاهان بر اساس مکان
- نمودار توزیع بر اساس نیاز آبی
- نمودار توزیع بر اساس نیاز نوری

## 💡 نکات مهم

- **حفظ اطلاعات**: اطلاعات در LocalStorage مرورگر ذخیره می‌شوند. پاک کردن حافظه مرورگر باعث حذف اطلاعات می‌شود
- **پشتیبان‌گیری**: به طور منظم از اطلاعات خود Export بگیرید
- **حجم تصاویر**: تصاویر به صورت Base64 ذخیره می‌شوند. استفاده از تصاویر با حجم کم توصیه می‌شود
- **مرورگرها**: برنامه با تمام مرورگرهای مدرن سازگار است (Chrome, Firefox, Edge, Safari)

## 🎨 سفارشی‌سازی

### تغییر رنگ‌ها

می‌توانید در فایل `assets/style.css` در بخش `:root` رنگ‌ها را تغییر دهید:

```css
:root {
    --primary-color: #2ecc71;      /* رنگ اصلی */
    --primary-dark: #27ae60;       /* رنگ تیره‌تر */
    --primary-light: #a8e6cf;      /* رنگ روشن‌تر */
    /* ... */
}
```
