# کدهای پلاک و پیش‌شماره استان‌های ایران

یک وب‌اپلیکیشن ساده و کاربردی برای مشاهده کدهای پلاک خودروها و پیش‌شماره تلفن ثابت استان‌های ایران، به همراه پیش‌شماره‌های تلفن همراه.

## ✨ ویژگی‌ها

- 🔍 **جستجوی پیشرفته**: جستجو بر اساس نام استان، شهر، کد پلاک یا پیش‌شماره
- 📱 **کاملاً ریسپانسیو**: نمایش بهینه در تمام دستگاه‌ها (دسکتاپ، تبلت، موبایل)
- 🎨 **طراحی مدرن و زیبا**: رابط کاربری جذاب با انیمیشن‌های نرم
- ⚡ **سریع و سبک**: بدون نیاز به سرور یا دیتابیس خارجی
- 🇮🇷 **پشتیبانی کامل از زبان فارسی**: با قابلیت RTL
- 📊 **اطلاعات جامع**: شامل ۳۱ استان ایران و تمام اپراتورهای موبایل

## 📋 اطلاعات موجود

### استان‌ها
برای هر استان اطلاعات زیر در دسترس است:
- کد پلاک خودرو
- پیش‌شماره تلفن ثابت
- لیست شهرهای مهم استان

### پیش‌شماره موبایل
پیش‌شماره‌های تمام اپراتورهای موبایل:
- همراه اول
- ایرانسل
- رایتل
- تله کیش
- اپراتورهای MVNO

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
این پروژه نیازی به نصب ابزار خاصی ندارد و فقط با یک مرورگر وب قابل اجرا است.

### نحوه استفاده

#### استفاده محلی
1. فایل‌های پروژه را دانلود کنید
2. فایل `index.html` را در مرورگر خود باز کنید

#### استفاده با سرور محلی
```bash
# با Python 3
python -m http.server 8000

# با Python 2
python -m SimpleHTTPServer 8000

# با Node.js (نیاز به نصب http-server)
npx http-server
```

سپس به آدرس `http://localhost:8000` در مرورگر بروید.

## 🌐 استقرار روی GitHub Pages

### مراحل استقرار:

1. **ایجاد مخزن GitHub**:
   - به GitHub بروید و یک مخزن جدید ایجاد کنید
   - نام مخزن را انتخاب کنید (مثلاً `iran-plate-codes`)

2. **آپلود فایل‌ها**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/username/iran-plate-codes.git
   git branch -M main
   git push -u origin main
   ```

3. **فعال‌سازی GitHub Pages**:
   - به تنظیمات (Settings) مخزن بروید
   - به بخش Pages بروید
   - در قسمت Source، شاخه `main` و پوشه `/ (root)` را انتخاب کنید
   - دکمه Save را بزنید

4. **دسترسی به سایت**:
   - پس از چند دقیقه، سایت شما در آدرس زیر در دسترس خواهد بود:
   - `https://username.github.io/iran-plate-codes/`

## 📁 ساختار پروژه

```
license-plate/
│
├── index.html              # فایل HTML اصلی
├── README.md              # مستندات پروژه
│
└── assets/                # پوشه دارایی‌ها
    ├── style.css          # استایل‌های CSS
    └── script.js          # کدهای JavaScript
```

## 🛠️ ساختار فایل‌ها

### index.html
فایل اصلی HTML که شامل ساختار صفحه است:
- هدر با عنوان و توضیحات
- بخش جستجو
- تب‌های استان‌ها و موبایل
- فوتر

### assets/style.css
استایل‌های کامل برنامه:
- طراحی ریسپانسیو
- انیمیشن‌ها و افکت‌های بصری
- Media queries برای موبایل و تبلت

### assets/script.js
منطق برنامه:
- داده‌های استان‌ها و موبایل
- عملکرد جستجو
- مدیریت تب‌ها
- رندر کردن کارت‌ها

## 🔄 نحوه به‌روزرسانی داده‌ها

برای افزودن یا ویرایش داده‌ها:

1. فایل `assets/script.js` را باز کنید
2. آرایه `provincesData` را برای استان‌ها ویرایش کنید
3. آرایه `mobileOperatorsData` را برای اپراتورها ویرایش کنید

مثال افزودن استان جدید:
```javascript
{
    name: 'نام استان',
    plateCode: 'کدهای پلاک',
    phoneCode: 'پیش‌شماره',
    cities: 'شهرهای مهم'
}
```
