# نقشه ذهنی

یک ابزار تحت وب برای ایجاد و مدیریت نقشه‌های ذهنی با قابلیت ذخیره‌سازی و بازیابی اطلاعات.

## ویژگی‌ها

### نودها
- ✨ افزودن نودهای جدید با کلیک ساده
- ✏️ ویرایش عنوان و توضیحات هر نود
- 🎨 انتخاب رنگ برای هر نود
- 🖱️ جابجایی نودها با درگ اند دراپ
- 🗑️ حذف نودها

### اتصالات
- 🔗 ایجاد اتصال بین نودها
- ➡️ نمایش خطوط منحنی با فلش جهت‌دار
- 🎯 جلوگیری از ایجاد اتصال تکراری
- 🧹 حذف خودکار اتصالات هنگام حذف نود

### ذخیره‌سازی و بازیابی
- 💾 ذخیره خودکار در مرورگر
- 📂 بارگذاری از حافظه مرورگر
- ⬇️ دانلود فایل JSON
- ⬆️ آپلود و وارد کردن فایل JSON
- 🗄️ پشتیبانی از localStorage

## نحوه استفاده

### افزودن نود
1. روی دکمه "➕ افزودن نود" کلیک کنید
2. نود جدید در محل تصادفی ایجاد می‌شود
3. روی آیکون ✏️ کلیک کنید تا نود را ویرایش کنید

### ویرایش نود
1. روی آیکون ✏️ در گوشه نود کلیک کنید
2. عنوان، توضیحات و رنگ را تغییر دهید
3. روی "ذخیره" کلیک کنید

### جابجایی نود
- نود را بگیرید و به محل دلخواه بکشید

### ایجاد اتصال
1. روی دکمه "🔗 اتصال" کلیک کنید تا حالت اتصال فعال شود
2. روی نود اول کلیک کنید
3. روی نود دوم کلیک کنید
4. اتصال ایجاد می‌شود

### حذف نود
- روی آیکون ✖️ در گوشه نود کلیک کنید

### ذخیره و بارگذاری
- **💾 ذخیره**: ذخیره نقشه در مرورگر
- **📂 بارگذاری**: بازیابی از مرورگر
- **⬇️ دانلود**: دانلود به صورت فایل JSON
- **⬆️ آپلود**: وارد کردن فایل JSON
- **🗑️ پاک کردن**: پاک کردن همه نودها و اتصالات

## نصب و راه‌اندازی

### استفاده محلی
1. فایل‌ها را دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید

### استقرار روی GitHub Pages
1. مخزن را fork یا clone کنید
2. به تنظیمات مخزن بروید
3. در بخش Pages، شاخه `main` و پوشه `root` را انتخاب کنید
4. لینک سایت در چند دقیقه آماده می‌شود

## مشارکت

برای مشارکت در این پروژه:

1. Fork کنید
2. شاخه جدید ایجاد کنید (`git checkout -b feature/amazing-feature`)
3. تغییرات را commit کنید (`git commit -m 'Add amazing feature'`)
4. به شاخه push کنید (`git push origin feature/amazing-feature`)
5. Pull Request ایجاد کنید
