# ⏰ مدیریت کرون جاب‌ها

یک برنامه وب کامل و حرفه‌ای برای مدیریت، ایجاد و نظارت بر کرون جاب‌های شما.

## 📋 ویژگی‌ها

### ✨ امکانات اصلی
- **ایجاد کرون جاب**: ایجاد کرون جاب‌های جدید با عبارات استاندارد cron
- **ویرایش و حذف**: مدیریت کامل جاب‌های موجود
- **فعال/غیرفعال کردن**: تغییر وضعیت جاب‌ها بدون حذف آن‌ها
- **پیش‌نمایش زنده**: مشاهده اجراهای بعدی هنگام تایپ عبارت کرون
- **نمایش اجراهای آینده**: مشاهده حداقل 10 اجرای بعدی با امکان افزایش
- **توضیح متنی**: توضیح خودکار عبارت کرون به زبان فارسی
- **تاریخچه فعالیت‌ها**: ثبت و نمایش تمام تغییرات
- **خروجی/ورودی**: دریافت و بارگذاری جاب‌ها در قالب JSON

### 🎨 طراحی و رابط کاربری
- **رسپانسیو**: طراحی کاملاً واکنش‌گرا برای موبایل، تبلت و دسکتاپ
- **انیمیشن‌های روان**: تجربه کاربری دلنشین با انیمیشن‌های زیبا
- **فونت فارسی**: استفاده از فونت Vazirmatn برای همه المان‌ها
- **RTL**: پشتیبانی کامل از راست به چپ
- **رنگ‌بندی مدرن**: استفاده از گرادیانت‌های زیبا و رنگ‌های متناسب
- **Toast Notifications**: نمایش پیام‌های موفقیت و خطا

### 💾 ذخیره‌سازی
- **LocalStorage**: ذخیره‌سازی خودکار در مرورگر
- **بدون نیاز به سرور**: تمام عملیات در سمت کاربر
- **ذخیره خودکار**: تمام تغییرات به صورت خودکار ذخیره می‌شوند
- **حفظ تاریخچه**: ذخیره تا 100 رویداد اخیر

## 📖 راهنمای عبارات Cron

### فرمت
```
* * * * *
│ │ │ │ │
│ │ │ │ └─── روز هفته (0-6، 0 = یکشنبه)
│ │ │ └───── ماه (1-12)
│ │ └─────── روز ماه (1-31)
│ └───────── ساعت (0-23)
└─────────── دقیقه (0-59)
```

### نمونه‌های رایج

| عبارت | توضیح |
|-------|--------|
| `* * * * *` | هر دقیقه |
| `0 * * * *` | هر ساعت |
| `0 0 * * *` | هر روز در نیمه‌شب |
| `0 9 * * *` | هر روز ساعت 9 صبح |
| `0 9 * * 1` | هر دوشنبه ساعت 9 صبح |
| `0 0 1 * *` | اول هر ماه |
| `*/5 * * * *` | هر 5 دقیقه |
| `0 9-17 * * *` | هر ساعت از 9 صبح تا 5 عصر |
| `0 9,12,15 * * *` | ساعت 9 و 12 و 15 |
| `0 9 * * 1-5` | دوشنبه تا جمعه ساعت 9 |

### نمادهای ویژه

- `*` : همه مقادیر ممکن
- `,` : لیست مقادیر (مثال: `1,3,5`)
- `-` : محدوده (مثال: `1-5`)
- `/` : گام (مثال: `*/5`)

## 🎯 امکانات پیشرفته

- نام دلخواه برای جاب
- وارد کردن عبارت cron
- توضیحات اختیاری
- انتخاب وضعیت اولیه (فعال/غیرفعال)
- پیش‌نمایش زنده اجراهای بعدی

### 2. مدیریت جاب‌ها
- مشاهده لیست تمام جاب‌ها
- ویرایش جاب‌های موجود
- حذف جاب‌های غیرضروری
- فعال/غیرفعال کردن سریع
- مشاهده زمان اجرای بعدی

- نمایش اولین 5 اجرای بعدی به صورت پیش‌فرض
- امکان افزایش تعداد نمایش با دکمه "نمایش بیشتر"
- نمایش تاریخ و زمان دقیق به فارسی

### 4. تاریخچه
- ثبت تمام عملیات (ایجاد، ویرایش، حذف، تغییر وضعیت)
- نمایش زمان دقیق هر عملیات
- امکان پاک کردن تاریخچه
- نگهداری حداکثر 100 رویداد اخیر

### 5. خروجی و ورودی
- دریافت خروجی JSON از تمام جاب‌ها
- بارگذاری جاب‌ها از فایل JSON
- پشتیبان‌گیری و بازیابی آسان
