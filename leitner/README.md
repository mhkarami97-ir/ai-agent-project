# 📚 سیستم لایتنر - تمرین واژگان

یک برنامه تحت وب برای یادگیری هوشمند واژگان با استفاده از روش سیستم لایتنر (فاصله‌گذاری تکرار).

## 🌟 ویژگی‌ها

### آزمون روزانه
- مرور کارت‌های فلش واژگان
- سیستم 5 جعبه‌ای لایتنر برای تکرار فاصله‌دار
- نمایش آمار لحظه‌ای (تعداد کل، باقیمانده، صحیح)
- پیام تبریک پس از اتمام آزمون روزانه

### افزودن کلمات
- افزودن تک‌به‌تک کلمات
- افزودن دسته‌ای کلمات از طریق فرمت `کلمه | معنی`
- اعلان موفقیت یا خطا

### لیست کلمات
- نمایش تمام کلمات اضافه شده
- فیلتر بر اساس جعبه (1 تا 5)
- جستجوی سریع
- امکان حذف کلمات
- نمایش تعداد کلمات در هر جعبه

### آمار
- تعداد کل کلمات
- تعداد مرور شده امروز
- روزهای متوالی یادگیری
- درصد دقت امروز
- نمودار توزیع کلمات در جعبه‌ها
- امکان پاک کردن تمام داده‌ها

## 🎯 نحوه کار سیستم لایتنر

سیستم لایتنر یک روش علمی برای یادگیری مبتنی بر تکرار فاصله‌دار است:

1. **جعبه 1**: کلمات جدید یا اشتباه - مرور روزانه
2. **جعبه 2**: کلمات که یک‌بار درست پاسخ داده‌اید - مرور هر 2 روز
3. **جعبه 3**: کلمات که دو‌بار متوالی درست پاسخ داده‌اید - مرور هر 4 روز
4. **جعبه 4**: کلمات که سه‌بار متوالی درست پاسخ داده‌اید - مرور هر 8 روز
5. **جعبه 5**: کلمات که چهار‌بار متوالی درست پاسخ داده‌اید - مرور هر 16 روز

### قوانین:
- ✅ **پاسخ صحیح**: کلمه به جعبه بالاتر منتقل می‌شود (حداکثر جعبه 5)
- ❌ **پاسخ نادرست**: کلمه به جعبه 1 برمی‌گردد

## 🚀 نصب و راه‌اندازی

### روش اول: اجرای مستقیم
1. فایل `index.html` را در مرورگر باز کنید
2. شروع به افزودن کلمات کنید!

### روش دوم: سرور محلی
```bash
# با Python 3
python -m http.server 8000

# با Node.js
npx serve

# با PHP
php -S localhost:8000
```

سپس به آدرس `http://localhost:8000` بروید.

### روش سوم: GitHub Pages
1. این مخزن را فورک کنید
2. به Settings > Pages بروید
3. Source را روی `main` branch تنظیم کنید
4. آدرس سایت شما: `https://username.github.io/leitner`

## 💾 ذخیره‌سازی داده‌ها

داده‌ها در مرورگر شما ذخیره می‌شوند و هیچ سروری نیاز نیست. این بدان معناست که:
- ✅ کاملاً خصوصی و آفلاین
- ✅ سرعت بالا
- ⚠️ پاک کردن cache مرورگر، داده‌ها را پاک می‌کند
- ⚠️ داده‌ها بین دستگاه‌های مختلف همگام‌سازی نمی‌شوند

### پشتیبان‌گیری دستی
برای پشتیبان‌گیری از داده‌ها:
1. Developer Tools را باز کنید (F12)
2. به Console بروید
3. دستور زیر را وارد کنید:
```javascript
copy(localStorage.getItem('leitnerData'))
```
4. داده‌ها در clipboard شما کپی می‌شوند - آن را در یک فایل ذخیره کنید

برای بازیابی:
1. Developer Tools > Console
2. دستور زیر را با جایگزینی `YOUR_BACKUP_DATA` اجرا کنید:
```javascript
localStorage.setItem('leitnerData', 'YOUR_BACKUP_DATA')
```
3. صفحه را رفرش کنید

## 📝 نحوه استفاده

### 1. افزودن کلمات
**تک‌به‌تک:**
- به تب "افزودن کلمات" بروید
- کلمه و معنی را وارد کنید
- دکمه "افزودن کلمه" را بزنید

**دسته‌ای:**
- به تب "افزودن کلمات" بروید
- در قسمت textarea، هر خط یک کلمه به فرمت: `کلمه | معنی`
- مثال:
  ```
  Apple | سیب
  Book | کتاب
  Cat | گربه
  ```
- دکمه "افزودن لیست" را بزنید

### 2. آزمون روزانه
- به تب "آزمون روزانه" بروید
- کلمه نمایش داده می‌شود
- سعی کنید معنی را به خاطر بیاورید
- دکمه "نمایش پاسخ" را بزنید
- بر اساس دانش خود "می‌دانستم" یا "نمی‌دانستم" را انتخاب کنید

### 3. مشاهده آمار
- به تب "آمار" بروید
- آمار کلی و توزیع جعبه‌ها را مشاهده کنید
- پیشرفت خود را دنبال کنید
