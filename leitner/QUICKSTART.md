# راهنمای سریع - سیستم لایتنر

## شروع سریع

### گام 1: باز کردن برنامه
- فایل `index.html` را در مرورگر خود باز کنید
- یا از یک سرور محلی استفاده کنید

### گام 2: افزودن کلمات اولیه
1. روی تب **"افزودن کلمات"** کلیک کنید
2. یک لیست کلمات آماده کنید (مثال زیر را کپی کنید):

```
Apple | سیب
Book | کتاب
Cat | گربه
Dog | سگ
House | خانه
Car | ماشین
Water | آب
Food | غذا
Friend | دوست
School | مدرسه
```

3. در قسمت "افزودن دسته‌ای"، متن بالا را paste کنید
4. دکمه **"افزودن لیست"** را بزنید

### گام 3: شروع آزمون
1. روی تب **"آزمون روزانه"** کلیک کنید
2. یک کلمه نمایش داده می‌شود
3. سعی کنید معنی را به خاطر بیاورید
4. **"نمایش پاسخ"** را بزنید
5. اگر می‌دانستید → **"✓ می‌دانستم"**
6. اگر نمی‌دانستید → **"✗ نمی‌دانستم"**

### گام 4: مشاهده پیشرفت
- تب **"آمار"** را باز کنید
- پیشرفت خود را دنبال کنید

## نکات مهم

✅ هر روز حداقل یک‌بار تمرین کنید
✅ صادق باشید در پاسخ‌ها
✅ به‌تدریج کلمات جدید اضافه کنید
✅ صبور باشید - یادگیری زمان می‌برد!

## آپلود به GitHub Pages

### گام 1: ایجاد مخزن
```bash
cd leitner
git init
git add .
git commit -m "Initial commit"
```

### گام 2: اتصال به GitHub
```bash
git remote add origin https://github.com/username/leitner.git
git branch -M main
git push -u origin main
```

### گام 3: فعال‌سازی GitHub Pages
1. به Settings مخزن بروید
2. بخش Pages را پیدا کنید
3. Source را روی `main` branch تنظیم کنید
4. Save کنید
5. بعد از چند دقیقه، سایت شما در `https://username.github.io/leitner` در دسترس خواهد بود

## پشتیبان‌گیری

### روش 1: Export دستی
1. F12 را بزنید (Developer Tools)
2. به Console بروید
3. این دستور را اجرا کنید:
```javascript
copy(localStorage.getItem('leitnerData'))
```
4. داده‌ها در clipboard شما است - در یک فایل txt ذخیره کنید

### روش 2: Import
1. Developer Tools > Console
2. این دستور را اجرا کنید (با جایگزینی BACKUP):
```javascript
localStorage.setItem('leitnerData', 'BACKUP_DATA_HERE')
```
3. صفحه را رفرش کنید

## سوالات متداول

**Q: چرا کلماتم پاک شد؟**
A: احتمالاً cache مرورگر پاک شده. از روش backup بالا استفاده کنید.

**Q: آیا می‌توانم بین دستگاه‌ها همگام کنم؟**
A: نه، اما می‌توانید با روش backup/restore داده‌ها را منتقل کنید.

**Q: چند جعبه وجود دارد؟**
A: 5 جعبه - هر جعبه فاصله زمانی متفاوتی دارد (1، 2، 4، 8، 16 روز)

**Q: اگر پاسخ اشتباه بدهم چه می‌شود؟**
A: کلمه به جعبه 1 برمی‌گردد و باید دوباره از اول شروع کنید.
