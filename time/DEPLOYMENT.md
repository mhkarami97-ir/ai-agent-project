## 🚀 استقرار روی GitHub Pages

### گام ۱: ایجاد مخزن در GitHub

1. به [GitHub](https://github.com) بروید و وارد شوید
2. روی دکمه **New** کلیک کنید تا مخزن جدید بسازید
3. نام مخزن را انتخاب کنید (مثلاً `timestamp-converter`)
4. مخزن را **Public** کنید
5. دکمه **Create repository** را بزنید

### گام ۲: آپلود فایل‌ها

در ترمینال PowerShell در پوشه پروژه دستورات زیر را اجرا کنید:

```powershell
# مقداردهی اولیه Git
git init

# اضافه کردن تمام فایل‌ها (پوشه .idea نادیده گرفته می‌شود)
git add .

# ایجاد اولین commit
git commit -m "Initial commit: Timestamp Converter"

# تغییر نام branch به main
git branch -M main

# اضافه کردن remote (آدرس مخزن GitHub خودتان)
git remote add origin https://github.com/USERNAME/REPO-NAME.git

# ارسال فایل‌ها به GitHub
git push -u origin main
```

**نکته**: `USERNAME` را با نام کاربری GitHub خود و `REPO-NAME` را با نام مخزن جایگزین کنید.

### گام ۳: فعال‌سازی GitHub Pages

1. به صفحه مخزن در GitHub بروید
2. روی تب **Settings** کلیک کنید
3. در منوی سمت چپ، گزینه **Pages** را انتخاب کنید
4. در قسمت **Source**:
   - Branch: `main` را انتخاب کنید
   - Folder: `/ (root)` را انتخاب کنید
5. روی **Save** کلیک کنید
6. چند دقیقه صبر کنید (معمولاً ۱-۵ دقیقه)

### گام ۴: دسترسی به سایت

سایت شما در آدرس زیر قابل دسترسی خواهد بود:
```
https://USERNAME.github.io/REPO-NAME/
```
