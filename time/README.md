# تبدیل‌کننده Timestamp (Unix ↔ تاریخ/زمان)

## 📋 توضیحات پروژه

این پروژه یک وب‌اپلیکیشن کامل و رسپانسیو برای تبدیل Unix Timestamp به تاریخ و زمان و بالعکس است. این ابزار به صورت کامل سمت کاربر (Client-Side) کار می‌کند و نیازی به سرور ندارد.

## ✨ امکانات

### 🕐 نمایش زمان فعلی
- **ساعت دیجیتال**: نمایش زمان و تاریخ فعلی با اعداد فارسی
- **ساعت آنالوگ**: نمایش گرافیکی زمان با عقربه‌های ساعت، دقیقه و ثانیه
- **Timestamp فعلی**: نمایش و کپی سریع timestamp جاری

### 🔄 تبدیل Unix به تاریخ
- تبدیل Unix timestamp به فرمت‌های مختلف:
  - زمان محلی
  - زمان UTC
  - تاریخ شمسی (فارسی)
  - فرمت ISO
  - زمان تهران
- دکمه "استفاده از زمان فعلی" برای تست سریع
- کپی سریع هر یک از نتایج

### 📅 تبدیل تاریخ به Unix
- انتخاب تاریخ و زمان دلخواه
- انتخاب منطقه زمانی (محلی یا UTC)
- نمایش timestamp در ثانیه و میلی‌ثانیه
- تنظیم خودکار تاریخ و زمان فعلی در ورودی‌ها

### 🌍 تبدیل بین مناطق زمانی
نمایش زنده زمان فعلی در ۸ منطقه زمانی مختلف:
- 🇮🇷 تهران (Asia/Tehran)
- 🌍 UTC
- 🇺🇸 نیویورک (America/New_York)
- 🇬🇧 لندن (Europe/London)
- 🇫🇷 پاریس (Europe/Paris)
- 🇦🇪 دبی (Asia/Dubai)
- 🇯🇵 توکیو (Asia/Tokyo)
- 🇦🇺 سیدنی (Australia/Sydney)

هر کارت شامل:
- نمایش زنده زمان
- نمایش timezone offset
- دکمه کپی سریع

### 📜 تاریخچه
- ذخیره خودکار تمام تبدیل‌ها در localStorage مرورگر
- نمایش ۲۰ تبدیل اخیر
- امکان پاک کردن تاریخچه

### 🎨 ویژگی‌های طراحی
- **رسپانسیو**: سازگار با موبایل، تبلت و دسکتاپ
- **RTL**: طراحی راست‌چین برای زبان فارسی
- **فونت Vazirmatn**: استفاده از فونت فارسی زیبا در تمام بخش‌ها
- **انیمیشن‌ها**: انیمیشن‌های نرم و چشم‌نواز
- **اعلان‌ها (Toast)**: اعلان‌های زیبا برای اعمال موفق
- **دارک گرادینت**: طراحی مدرن با رنگ‌های بنفش

## 📝 راهنمای استفاده

### تبدیل Unix به تاریخ
1. عدد timestamp را در کادر مربوطه وارد کنید
2. دکمه "تبدیل" را بزنید
3. نتایج در فرمت‌های مختلف نمایش داده می‌شود
4. برای کپی هر نتیجه، روی دکمه "کپی" کنار آن کلیک کنید

### تبدیل تاریخ به Unix
1. تاریخ دلخواه را انتخاب کنید
2. زمان را وارد کنید
3. منطقه زمانی (محلی یا UTC) را انتخاب کنید
4. دکمه "تبدیل به Unix" را بزنید

### مشاهده زمان در کشورهای مختلف
زمان فعلی در ۸ کشور مختلف به صورت زنده نمایش داده می‌شود. برای کپی زمان هر کشور، روی دکمه "کپی" در کارت مربوطه کلیک کنید.

## 🛠️ توسعه و سفارشی‌سازی

### اضافه کردن منطقه زمانی جدید
در فایل `index.html`، یک کارت جدید اضافه کنید:

```html
<div class="timezone-card" data-timezone="TIMEZONE_NAME">
    <h3>🌍 نام شهر</h3>
    <div class="timezone-time" data-tz-display="TIMEZONE_NAME">--:--:--</div>
    <div class="timezone-offset" data-tz-offset="TIMEZONE_NAME">UTC +0:00</div>
    <button class="copy-btn-small" data-copy-tz="TIMEZONE_NAME">کپی</button>
</div>
```

### تغییر رنگ‌ها
رنگ‌های اصلی در فایل `assets/style.css` قابل تغییر هستند:

```css
/* رنگ اصلی */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* رنگ دکمه‌ها */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
