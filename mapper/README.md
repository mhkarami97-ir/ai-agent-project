# تبدیل واحد

یک وب‌اپلیکیشن کاربردی برای تبدیل واحدهای مختلف اندازه‌گیری با قابلیت ذخیره علاقه‌مندی‌ها.

## ویژگی‌ها

### 🔄 تبدیل واحدهای متنوع
- **طول**: متر، کیلومتر، سانتی‌متر، میلی‌متر، مایل، یارد، فوت، اینچ
- **وزن**: کیلوگرم، گرم، میلی‌گرم، تن، پاند، اونس
- **دما**: سلسیوس، فارنهایت، کلوین
- **سرعت**: متر بر ثانیه، کیلومتر بر ساعت، مایل بر ساعت، گره دریایی، فوت بر ثانیه

### ⭐ علاقه‌مندی‌ها
- ذخیره تبدیل‌های پرکاربرد
- بارگذاری سریع تبدیل‌های ذخیره شده
- حذف موارد دلخواه
- پاک کردن همه موارد با یک کلیک

### 🎨 طراحی و رابط کاربری
- طراحی مدرن و جذاب
- رسپانسیو و سازگار با موبایل، تبلت و دسکتاپ
- پشتیبانی کامل از زبان فارسی و راست‌چین
- انیمیشن‌های روان
- رنگ‌بندی زیبا و چشم‌نواز

### 💾 ذخیره‌سازی محلی
- ذخیره خودکار علاقه‌مندی‌ها در مرورگر
- دسترسی به داده‌ها بدون نیاز به اتصال اینترنت
- حفظ داده‌ها حتی پس از بستن مرورگر

### ⚡ امکانات اضافی
- تبدیل آنی در حین تایپ
- دکمه تعویض سریع واحدها
- فرمت هوشمند اعداد
- پشتیبانی از اعداد اعشاری و بزرگ
- نوتیفیکیشن‌های زیبا برای عملیات موفق

## نحوه استفاده

### 1. تبدیل واحد
1. دسته‌بندی مورد نظر (طول، وزن، دما، سرعت) را انتخاب کنید
2. واحد مبدأ را از لیست انتخاب کنید
3. عدد مورد نظر را وارد کنید
4. واحد مقصد را انتخاب کنید
5. نتیجه به صورت خودکار نمایش داده می‌شود
6. می‌توانید دکمه "تبدیل" را برای اطمینان بیشتر کلیک کنید

### 2. ذخیره در علاقه‌مندی‌ها
1. پس از انجام تبدیل، دکمه "ذخیره در علاقه‌مندی‌ها" را کلیک کنید
2. تبدیل انجام شده در بخش علاقه‌مندی‌ها ذخیره می‌شود

### 3. استفاده از علاقه‌مندی‌ها
- روی دکمه "بارگذاری" کلیک کنید تا تبدیل ذخیره شده بارگذاری شود
- برای حذف یک مورد، دکمه "حذف" را کلیک کنید
- برای پاک کردن همه موارد، دکمه "پاک کردن همه" را کلیک کنید

### 4. تعویض واحدها
- روی دکمه تعویض (آیکون دو فلش) کلیک کنید تا واحد مبدأ و مقصد جابجا شوند

## ساختار پروژه

```
mapper/
├── index.html          # صفحه اصلی
├── assets/
│   ├── style.css      # استایل‌های CSS
│   └── script.js      # منطق و عملکرد JavaScript
└── README.md          # مستندات پروژه
```

## فایل‌های پروژه

### index.html
فایل HTML اصلی شامل ساختار صفحه، فرم تبدیل، دسته‌بندی‌ها و بخش علاقه‌مندی‌ها

### assets/style.css
شامل تمام استایل‌های CSS، طراحی رسپانسیو، رنگ‌بندی، انیمیشن‌ها و فونت Vazirmatn

### assets/script.js
شامل تمام منطق برنامه:
- تبدیل واحدها با فرمول‌های دقیق
- مدیریت علاقه‌مندی‌ها
- ذخیره‌سازی در LocalStorage
- مدیریت رویدادها و تعاملات کاربر

## نصب و راه‌اندازی

### روش 1: استفاده مستقیم
فایل `index.html` را در مرورگر باز کنید.

### روش 2: استفاده با سرور محلی
```bash
# استفاده از Python
python -m http.server 8000

# یا استفاده از Node.js
npx http-server
```

سپس به آدرس `http://localhost:8000` بروید.

### روش 3: استقرار در GitHub Pages

1. پروژه را در یک مخزن GitHub بارگذاری کنید
2. به تنظیمات مخزن بروید
3. به بخش "Pages" بروید
4. شاخه `main` و پوشه `root` را انتخاب کنید
5. روی "Save" کلیک کنید
6. پس از چند دقیقه، سایت شما در `https://username.github.io/repository-name` در دسترس خواهد بود
