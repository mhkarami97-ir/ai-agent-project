# افزودن واترمارک به تصویر

یک ابزار آنلاین و رایگان برای افزودن واترمارک متنی یا تصویری به تصاویر شما.

## ویژگی‌های اصلی

- ✨ **واترمارک متنی**: افزودن متن دلخواه با کنترل کامل بر اندازه، رنگ و موقعیت
- 🖼️ **واترمارک تصویری**: افزودن لوگو یا تصویر با امکان تغییر اندازه
- 🎨 **کنترل شفافیت**: تنظیم میزان شفافیت واترمارک از 0 تا 100 درصد
- 📍 **انتخاب موقعیت**: 9 موقعیت مختلف برای قرارگیری واترمارک
- 🔄 **چرخش**: امکان چرخاندن واترمارک از -180 تا 180 درجه
- 💾 **ذخیره تنظیمات**: ذخیره خودکار تنظیمات برای استفاده‌های بعدی
- 📱 **طراحی واکنش‌گرا**: سازگار با موبایل، تبلت و دسکتاپ
- 🇮🇷 **زبان فارسی**: رابط کاربری کاملاً فارسی با فونت Vazirmatn

## نحوه استفاده

### 1. بارگذاری تصویر
- روی کادر بارگذاری کلیک کنید یا تصویر را به داخل آن بکشید
- فرمت‌های پشتیبانی شده: JPG، PNG، GIF

### 2. انتخاب نوع واترمارک

#### واترمارک متنی
- متن دلخواه خود را وارد کنید
- اندازه فونت را تنظیم کنید (12-200 پیکسل)
- رنگ متن را انتخاب کنید

#### واترمارک تصویری
- روی دکمه "انتخاب لوگو" کلیک کنید
- فایل تصویری لوگو را انتخاب کنید
- اندازه لوگو را تنظیم کنید (20-500 پیکسل)

### 3. تنظیمات مشترک
- **شفافیت**: میزان شفافیت واترمارک (0-100%)
- **موقعیت**: یکی از 9 موقعیت مختلف را انتخاب کنید
- **چرخش**: زاویه چرخش واترمارک (-180 تا 180 درجه)

### 4. دانلود
- روی دکمه "دانلود تصویر" کلیک کنید
- تصویر با واترمارک به صورت PNG ذخیره می‌شود

## تکنولوژی‌ها

این پروژه با استفاده از تکنولوژی‌های زیر ساخته شده است:

- **HTML5**: ساختار صفحه
- **CSS3**: استایل‌دهی و طراحی واکنش‌گرا
- **JavaScript (Vanilla)**: منطق برنامه و کار با Canvas API
- **Canvas API**: پردازش تصاویر و افزودن واترمارک
- **Local Storage API**: ذخیره تنظیمات کاربر
- **Vazirmatn Font**: فونت فارسی زیبا و خوانا

## ویژگی‌های فنی

تمام پردازش‌ها در مرورگر شما انجام می‌شود و هیچ فایلی به سرور ارسال نمی‌شود. این به معنای:
- 🔒 حفظ حریم خصوصی شما
- ⚡ سرعت بالای پردازش
- 🌐 عدم نیاز به اتصال اینترنت پس از بارگذاری اولیه

### ذخیره‌سازی محلی
تنظیمات شما به صورت خودکار در Local Storage مرورگر ذخیره می‌شود و در بازدید بعدی بازیابی می‌گردد.

### سازگاری با مرورگرها
- Chrome/Edge (نسخه 90+)
- Firefox (نسخه 88+)
- Safari (نسخه 14+)
- Opera (نسخه 76+)

## نصب و راه‌اندازی

### استفاده آنلاین
این برنامه برای استفاده در GitHub Pages آماده است.

### اجرای محلی
1. این مخزن را کلون کنید:
```bash
git clone https://github.com/username/water-mark.git
```

2. وارد پوشه پروژه شوید:
```bash
cd water-mark
```

3. فایل `index.html` را در مرورگر باز کنید یا از یک سرور محلی استفاده کنید:
```bash
# با Python 3
python -m http.server 8000

# با Node.js (http-server)
npx http-server
```

4. مرورگر را باز کرده و به آدرس زیر بروید:
```
http://localhost:8000
```

## استقرار در GitHub Pages

1. مخزن را در GitHub ایجاد کنید
2. فایل‌های پروژه را push کنید
3. در تنظیمات مخزن به بخش "Pages" بروید
4. منبع را روی `main` branch تنظیم کنید
5. صبر کنید تا GitHub Pages فعال شود
