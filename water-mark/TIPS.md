# نکات و ترفندها

راهنمای استفاده بهتر از ابزار افزودن واترمارک

## 💡 نکات عمومی

### 1. انتخاب تصویر مناسب
- **کیفیت**: از تصاویر با کیفیت بالا استفاده کنید
- **فرمت**: JPG برای عکس‌ها، PNG برای تصاویر با پس‌زمینه شفاف
- **حجم**: تصاویر خیلی بزرگ ممکن است کند پردازش شوند

### 2. واترمارک متنی موثر
- **خوانایی**: از رنگ‌هایی استفاده کنید که با تصویر کنتراست داشته باشند
- **اندازه**: واترمارک نباید خیلی بزرگ یا خیلی کوچک باشد
- **موقعیت**: معمولاً گوشه‌های تصویر بهترین مکان هستند

### 3. واترمارک تصویری (لوگو)
- **شفافیت**: لوگوهای PNG با پس‌زمینه شفاف بهتر هستند
- **اندازه**: لوگو نباید بیش از 20% تصویر را بپوشاند
- **کیفیت**: از لوگوهای با کیفیت بالا استفاده کنید

## 🎨 بهترین تنظیمات

### برای عکس‌های شخصی
```
نوع: متن
متن: © نام شما 2025
اندازه فونت: 36-48
رنگ: سفید یا مشکی (بسته به رنگ عکس)
شفافیت: 60-80%
موقعیت: پایین راست یا پایین چپ
چرخش: 0°
```

### برای محصولات تجاری
```
نوع: تصویر (لوگو)
اندازه لوگو: 80-120
شفافیت: 50-70%
موقعیت: گوشه یا مرکز
چرخش: 0° یا -45° (برای پوشش بیشتر)
```

### برای اسناد و مدارک
```
نوع: متن
متن: محرمانه / CONFIDENTIAL
اندازه فونت: 80-120
رنگ: قرمز یا خاکستری
شفافیت: 30-50%
موقعیت: مرکز
چرخش: -45°
```

## ⚙️ ترفندهای پیشرفته

### 1. واترمارک نامرئی
برای حفاظت بدون تخریب زیبایی تصویر:
- شفافیت: 20-30%
- رنگ: خاکستری روشن
- موقعیت: مرکز یا گوشه

### 2. واترمارک برجسته
برای جلوگیری از کپی‌برداری:
- شفافیت: 70-90%
- اندازه: بزرگ
- موقعیت: مرکز
- چرخش: -30° تا -45°

### 3. واترمارک حرفه‌ای
برای برند شخصی:
- استفاده از لوگو + متن
- رنگ‌های برند
- موقعیت ثابت در همه تصاویر
- شفافیت 50-60%

## 🚀 میانبرهای کیبورد (در آینده)

در نسخه‌های بعدی می‌توانیم اضافه کنیم:
- `Ctrl/Cmd + O`: باز کردن تصویر
- `Ctrl/Cmd + S`: دانلود تصویر
- `Ctrl/Cmd + Z`: بازگشت به تصویر اصلی
- `Arrow Keys`: جابجایی واترمارک
- `+/-`: بزرگ/کوچک کردن واترمارک

## 📋 چک‌لیست قبل از دانلود

- [ ] واترمارک زیبایی تصویر را خراب نکرده؟
- [ ] اندازه واترمارک مناسب است؟

## 🔧 رفع مشکلات رایج

### مشکل: واترمارک دیده نمی‌شود
**راه‌حل**:
- شفافیت را افزایش دهید
- رنگ را تغییر دهید (کنتراست بیشتر)
- اندازه را بزرگ‌تر کنید

### مشکل: واترمارک خیلی درشت است
**راه‌حل**:
- اندازه فونت یا لوگو را کاهش دهید
- از موقعیت گوشه به جای مرکز استفاده کنید

### مشکل: کیفیت تصویر پایین آمده
**راه‌حل**:
- از تصویر اصلی با کیفیت بالاتر استفاده کنید
- فایل PNG به جای JPG ذخیره می‌شود (کیفیت بالاتر)

### مشکل: فونت فارسی نمایش داده نمی‌شود
**راه‌حل**:
- مطمئن شوید اتصال اینترنت برای بارگذاری فونت وجود دارد
- کش مرورگر را پاک کنید
- صفحه را رفرش کنید (F5)

## 🎓 موارد استفاده

### عکاسی حرفه‌ای
- واترمارک لوگو در گوشه
- شفافیت 60-70%
- اندازه متوسط

### رسانه‌های اجتماعی
- واترمارک متنی یا لوگو
- موقعیت گوشه پایین
- شفافیت 50-60%

### فروش آنلاین
- واترمارک مرکزی با شفافیت بالا
- جلوگیری از کپی
- چرخش -45 درجه

### آموزشی و مستندات
- واترمارک متنی واضح
- شامل تاریخ یا شماره نسخه
- گوشه بالا یا پایین

## 🌟 ترکیب‌های رنگی پیشنهادی

### تصاویر روشن
- متن مشکی با شفافیت 60-70%
- متن آبی تیره با شفافیت 50-60%
- متن خاکستری تیره با شفافیت 70-80%

### تصاویر تیره
- متن سفید با شفافیت 70-80%
- متن زرد روشن با شفافیت 60-70%
- متن آبی روشن با شفافیت 60-70%

### تصاویر رنگارنگ
- متن سفید یا مشکی (بسته به پس‌زمینه غالب)
- استفاده از سایه متن (خودکار اعمال می‌شود)
- شفافیت 70-80%
