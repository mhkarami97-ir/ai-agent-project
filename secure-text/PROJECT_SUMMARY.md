### 🚀 نحوه استفاده

#### تست محلی

1. **روش اول - Python:**
```bash
cd D:\Local\my\ai-agent-project\secure-text
python -m http.server 8000
```
سپس به `http://localhost:8000` بروید

2. **روش دوم - Node.js:**
```bash
cd D:\Local\my\ai-agent-project\secure-text
npx http-server
```

3. **روش سوم - مستقیم:**
فایل `index.html` را در مرورگر باز کنید (برخی امکانات ممکن است محدود باشند)

#### استقرار در GitHub Pages

1. یک مخزن GitHub ایجاد کنید
2. فایل‌ها را push کنید
3. GitHub Pages را در تنظیمات فعال کنید
4. به `https://yourusername.github.io/secure-text/` بروید

برای جزئیات بیشتر، فایل `DEPLOYMENT.md` را مطالعه کنید.
