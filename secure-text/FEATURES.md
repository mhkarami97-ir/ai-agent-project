# 📋 لیست کامل امکانات

## 🔐 امنیت و رمزنگاری

### رمزنگاری سمت کلاینت
- **رمزنگاری XOR با Base64**: تمام یادداشت‌ها قبل از ذخیره‌سازی رمزنگاری می‌شوند
- **هش SHA-256**: رمز عبور با استفاده از Web Crypto API هش می‌شود
- **بدون ارسال داده**: تمام عملیات در مرورگر انجام می‌شود
- **حفاظت از رمز عبور**: امکان فراموشی رمز عبور وجود دارد اما بازیابی غیرممکن است

### قفل خودکار
- **زمان‌سنج قابل تنظیم**: از 1 تا 60 دقیقه یا غیرفعال
- **تشخیص فعالیت**: حرکت ماوس، فشار کلید، اسکرول، کلیک
- **نمایش زمان باقی‌مانده**: نمایش آنی زمان تا قفل شدن
- **قفل دستی**: دکمه قفل فوری در هدر

## 📝 مدیریت یادداشت‌ها

### ایجاد و ویرایش
- **ایجاد سریع**: دکمه یادداشت جدید در سایدبار
- **ویرایش عنوان**: فیلد ورودی جداگانه برای عنوان
- **ویرایشگر متن**: Textarea بزرگ برای محتوای یادداشت
- **ذخیره دستی**: دکمه ذخیره با بازخورد بصری
- **شمارنده کاراکتر**: نمایش تعداد کاراکترهای متن

### نمایش و مرور
- **لیست یادداشت‌ها**: نمایش تمام یادداشت‌ها در سایدبار
- **پیش‌نمایش**: نمایش 50 کاراکتر اول محتوا
- **تاریخ**: نمایش تاریخ آخرین تغییر به صورت شمسی
- **یادداشت فعال**: هایلایت یادداشت انتخاب‌شده
- **شمارش**: نمایش تعداد کل یادداشت‌ها

### جستجو
- **جستجوی لحظه‌ای**: جستجو در حین تایپ
- **جستجو در عنوان**: پیدا کردن یادداشت‌ها با عنوان
- **جستجو در محتوا**: پیدا کردن یادداشت‌ها با محتوا
- **فیلتر خودکار**: لیست یادداشت‌ها به صورت خودکار فیلتر می‌شود

### حذف
- **حذف ایمن**: تأیید قبل از حذف
- **حذف فوری**: حذف یادداشت با یک کلیک پس از تأیید

## ⚙️ تنظیمات

### قفل خودکار
- **انتخاب زمان**: 1، 2، 5، 10، 15، 30، 60 دقیقه یا غیرفعال
- **ذخیره خودکار**: تنظیمات به صورت خودکار ذخیره می‌شود

### تغییر رمز عبور
- **اعتبارسنجی**: بررسی رمز عبور فعلی
- **رمزنگاری مجدد**: تمام یادداشت‌ها با رمز جدید رمزنگاری می‌شوند
- **تایید**: تأیید رمز عبور جدید برای جلوگیری از اشتباه

### حذف کامل
- **بازنشانی برنامه**: حذف تمام یادداشت‌ها و تنظیمات
- **تأیید دوباره**: پیام هشدار قبل از حذف
- **بدون بازگشت**: عملیات قابل بازگشت نیست

## 🎨 طراحی و رابط کاربری

### تم تیره
- **پالت رنگی مدرن**: ترکیب آبی، بنفش و خاکستری
- **کنتراست بالا**: خوانایی عالی در شب
- **سایه‌ها**: سایه‌های نرم برای عمق

### فونت فارسی
- **Vazirmatn**: فونت مدرن و زیبا برای فارسی
- **RTL**: طراحی کامل راست به چپ
- **اندازه‌های مختلف**: از 0.75rem تا 2.5rem

### رسپانسیو
- **دسکتاپ**: لایوت دو ستونه (سایدبار + ویرایشگر)
- **تبلت**: سایدبار کوچک‌تر
- **موبایل**: لایوت تک ستونه، سایدبار قابل جمع شدن
- **بریک پوینت‌ها**: 1024px، 768px، 480px

### انیمیشن‌ها
- **ترانزیشن‌های نرم**: 0.3 ثانیه برای تمام تغییرات
- **Hover Effects**: تغییر رنگ و جابجایی روی دکمه‌ها
- **فید این**: انیمیشن ظاهر شدن یادداشت‌ها
- **تبدیل**: انیمیشن بستن مودال‌ها

## 🔍 جزئیات فنی

### ذخیره‌سازی
- **LocalStorage**: ذخیره داده‌ها در مرورگر
- **JSON**: فرمت داده‌ها
- **رمزنگاری**: تبدیل به Base64 پس از رمزنگاری

### رمزنگاری
- **XOR Cipher**: الگوریتم رمزنگاری متقارن
- **SHA-256**: هش رمز عبور
- **Base64**: انکد کردن داده‌های رمزشده

### مدیریت حالت
- **کلاس‌های ES6**: ساختار شی‌گرا
- **Managers**: CryptoManager، NotesManager، AutoLockManager
- **Event Driven**: مدیریت رویدادها با addEventListener

### سازگاری
- **Modern Browsers**: Chrome، Firefox، Safari، Edge
- **Web APIs**: Crypto API، LocalStorage API
- **ES6+**: استفاده از قابلیت‌های جدید JavaScript
