# 🔒 امن‌ساز یادداشت‌ها

یک برنامه وب امن برای نگهداری یادداشت‌ها با رمزنگاری سمت کلاینت و قابلیت قفل خودکار.

## ✨ امکانات

- **🔐 رمزنگاری سمت کلاینت**: تمام یادداشت‌ها با استفاده از رمز عبور شما رمزنگاری می‌شوند
- **⏱️ قفل خودکار**: برنامه پس از مدت زمان مشخص بدون فعالیت، به طور خودکار قفل می‌شود
- **💾 ذخیره‌سازی محلی**: داده‌ها در مرورگر شما ذخیره می‌شوند
- **🔍 جستجو**: جستجوی سریع در عنوان و محتوای یادداشت‌ها
- **📱 رسپانسیو**: طراحی کاملاً واکنش‌گرا برای موبایل، تبلت و دسکتاپ
- **🌙 طراحی تیره**: رابط کاربری مدرن با تم تیره
- **✍️ ویرایشگر ساده**: ایجاد، ویرایش و حذف یادداشت‌ها
- **🔄 تغییر رمز عبور**: امکان تغییر رمز عبور در هر زمان

## 🚀 نحوه استفاده

### استفاده آنلاین

برنامه را می‌توانید به صورت مستقیم از GitHub Pages استفاده کنید:

[https://yourusername.github.io/secure-text](https://yourusername.github.io/secure-text)

### استفاده محلی

1. مخزن را کلون کنید یا فایل‌ها را دانلود کنید:
```bash
git clone https://github.com/yourusername/secure-text.git
cd secure-text
```

2. فایل `index.html` را در مرورگر خود باز کنید

یا می‌توانید یک سرور محلی ساده راه‌اندازی کنید:

```bash
# با Python 3
python -m http.server 8000

# یا با Node.js (npx http-server)
npx http-server
```

سپس به `http://localhost:8000` بروید

## 📖 راهنمای استفاده

### اولین بار

1. هنگام ورود اول، یک رمز عبور قوی ایجاد کنید
2. این رمز عبور برای رمزنگاری تمام یادداشت‌های شما استفاده می‌شود
3. **مهم**: رمز عبور خود را فراموش نکنید! در صورت فراموشی، امکان بازیابی یادداشت‌ها وجود ندارد

### ایجاد یادداشت

1. روی دکمه "➕ یادداشت جدید" کلیک کنید
2. عنوان و محتوای یادداشت را وارد کنید
3. روی دکمه "💾 ذخیره" کلیک کنید

### جستجو

از کادر جستجو در سمت راست استفاده کنید تا یادداشت‌های مورد نظر خود را پیدا کنید.

### قفل خودکار

- برنامه پس از مدت زمان مشخص (پیش‌فرض: 5 دقیقه) بدون فعالیت، به طور خودکار قفل می‌شود
- می‌توانید زمان قفل خودکار را از تنظیمات تغییر دهید
- برای ورود مجدد، باید رمز عبور را وارد کنید

### تنظیمات

از دکمه تنظیمات (⚙️) در هدر می‌توانید:
- زمان قفل خودکار را تغییر دهید
- رمز عبور خود را تغییر دهید
- تمام داده‌ها را حذف کنید

## 🔒 امنیت

- **رمزنگاری سمت کلاینت**: تمام عملیات رمزنگاری در مرورگر شما انجام می‌شود
- **بدون سرور**: هیچ داده‌ای به سرور ارسال نمی‌شود
- **ذخیره‌سازی محلی**: داده‌ها فقط در مرورگر شما ذخیره می‌شوند
- **هش رمز عبور**: رمز عبور با استفاده از SHA-256 هش می‌شود
- **قفل خودکار**: حفاظت از یادداشت‌ها در صورت عدم فعالیت

## ⚠️ نکات مهم

1. **رمز عبور را فراموش نکنید**: هیچ راهی برای بازیابی یادداشت‌ها در صورت فراموشی رمز عبور وجود ندارد
2. **پشتیبان‌گیری**: برای ایمنی بیشتر، از یادداشت‌های مهم خود نسخه پشتیبان تهیه کنید
3. **حریم خصوصی**: داده‌ها در مرورگر شما ذخیره می‌شوند و در صورت پاک کردن داده‌های مرورگر، حذف خواهند شد
4. **رمز عبور قوی**: از رمز عبور قوی با حداقل 8 کاراکتر شامل حروف، اعداد و نمادها استفاده کنید
