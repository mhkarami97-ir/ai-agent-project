# 🚀 راهنمای استقرار در GitHub Pages

این راهنما به شما کمک می‌کند تا برنامه امن‌ساز یادداشت‌ها را در GitHub Pages مستقر کنید.

## مراحل استقرار

### 1. ایجاد مخزن در GitHub

1. وارد حساب GitHub خود شوید
2. روی دکمه "New repository" کلیک کنید
3. یک نام برای مخزن انتخاب کنید (مثلاً `secure-text`)
4. مخزن را Public یا Private ایجاد کنید
5. روی "Create repository" کلیک کنید

### 2. آپلود کردن فایل‌ها

#### روش اول: استفاده از Git Command Line

```bash
# رفتن به پوشه پروژه
cd secure-text

# مقداردهی اولیه Git
git init

# افزودن فایل‌ها
git add .

# ایجاد اولین Commit
git commit -m "Initial commit: Secure Notes App"

# افزودن Remote (آدرس مخزن خود را جایگزین کنید)
git remote add origin https://github.com/yourusername/secure-text.git

# Push کردن فایل‌ها
git branch -M main
git push -u origin main
```

#### روش دوم: آپلود از طریق رابط وب GitHub

1. وارد مخزن خود شوید
2. روی "uploading an existing file" کلیک کنید
3. تمام فایل‌ها را بکشید و رها کنید (Drag & Drop)
4. روی "Commit changes" کلیک کنید

### 3. فعال‌سازی GitHub Pages

1. در مخزن خود، به تب "Settings" بروید
2. در منوی سمت راست، روی "Pages" کلیک کنید
3. در قسمت "Source"، گزینه "Deploy from a branch" را انتخاب کنید
4. Branch را روی `main` و پوشه را روی `/ (root)` تنظیم کنید
5. روی "Save" کلیک کنید

### 4. دسترسی به برنامه

پس از چند دقیقه، برنامه شما در آدرس زیر در دسترس خواهد بود:

```
https://yourusername.github.io/secure-text/
```

نام کاربری خود را در آدرس بالا جایگزین کنید.

## ✅ تست برنامه

1. به آدرس GitHub Pages خود بروید
2. رمز عبوری ایجاد کنید
3. یادداشت جدیدی ایجاد کنید
4. قابلیت قفل خودکار و سایر امکانات را امتحان کنید

## 🔄 به‌روزرسانی برنامه

برای به‌روزرسانی برنامه، تغییرات خود را Commit و Push کنید:

```bash
git add .
git commit -m "Update: توضیحات تغییرات"
git push origin main
```

GitHub Pages به طور خودکار برنامه را به‌روزرسانی می‌کند (ممکن است چند دقیقه طول بکشد).

## 🌐 دامنه سفارشی (اختیاری)

اگر می‌خواهید از دامنه سفارشی خود استفاده کنید:

1. در تنظیمات GitHub Pages، یک فایل `CNAME` ایجاد کنید
2. دامنه خود را وارد کنید (مثلاً `notes.example.com`)
3. در تنظیمات DNS دامنه خود، یک رکورد CNAME به `yourusername.github.io` اضافه کنید
