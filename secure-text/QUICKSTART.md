# 🚀 راهنمای سریع شروع

## شروع سریع (5 دقیقه)

### 1️⃣ تست محلی

**ساده‌ترین روش** - فایل `index.html` را با مرورگر باز کنید:
```
D:\Local\my\ai-agent-project\secure-text\index.html
```

یا با استفاده از سرور محلی:
```bash
cd D:\Local\my\ai-agent-project\secure-text
python -m http.server 8000
```
سپس: http://localhost:8000

### 2️⃣ استقرار در GitHub Pages

#### گام 1: ایجاد مخزن
```bash
cd D:\Local\my\ai-agent-project\secure-text
git init
git add .
git commit -m "Initial commit"
```

#### گام 2: ارسال به GitHub
```bash
git remote add origin https://github.com/USERNAME/secure-text.git
git branch -M main
git push -u origin main
```

#### گام 3: فعال‌سازی Pages
1. Settings → Pages
2. Source: Deploy from branch
3. Branch: main / (root)
4. Save

✅ آماده! برنامه شما در `https://USERNAME.github.io/secure-text` در دسترس است

### 3️⃣ اولین استفاده

1. **باز کردن برنامه** - به آدرس برنامه بروید
2. **ایجاد رمز عبور** - یک رمز عبور قوی انتخاب کنید
3. **ایجاد یادداشت** - روی "➕ یادداشت جدید" کلیک کنید
4. **نوشتن** - عنوان و محتوا را وارد کنید
5. **ذخیره** - روی "💾 ذخیره" کلیک کنید

### 4️⃣ قفل خودکار

- برنامه پس از 5 دقیقه بدون فعالیت قفل می‌شود
- می‌توانید از تنظیمات (⚙️) زمان را تغییر دهید
- برای قفل دستی از دکمه 🔒 استفاده کنید

## ❓ سوالات متداول

**Q: رمز عبور را فراموش کردم چه کنم؟**  
A: متاسفانه بازیابی امکان‌پذیر نیست. باید تمام داده‌ها را حذف کنید.

**Q: آیا داده‌هایم امن هستند؟**  
A: بله! همه چیز در مرورگر شما رمزنگاری می‌شود و هیچ داده‌ای ارسال نمی‌شود.

**Q: آیا به اینترنت نیاز دارم؟**  
A: فقط برای بارگذاری فونت. بعد از بار اول می‌توانید آفلاین استفاده کنید.

**Q: چگونه پشتیبان تهیه کنم؟**  
A: یادداشت‌های خود را کپی کنید یا از Developer Tools → Application → LocalStorage استفاده کنید.
