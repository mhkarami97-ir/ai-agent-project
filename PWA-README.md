# 🛠️ ابزارهای وب - Web Tools Collection

مجموعه کاملی از 46+ ابزار آنلاین ساخته شده با هوش مصنوعی که به صورت PWA (Progressive Web App) قابل نصب روی موبایل و دسکتاپ است.

## ✨ ویژگی‌های اصلی

### 🌐 Progressive Web App (PWA)
- **قابل نصب**: روی موبایل (Android/iOS) و دسکتاپ (Windows/Mac/Linux)
- **آفلاین**: کار کردن بدون اینترنت با Service Worker
- **سریع**: کش کردن فایل‌ها برای بارگذاری فوری
- **رسپانسیو**: طراحی زیبا در تمام دستگاه‌ها

### 🎨 رابط کاربری
- **دو زبانه**: فارسی و انگلیسی همزمان
- **جستجوی لحظه‌ای**: پیدا کردن سریع ابزار مورد نظر
- **Dark/Light Mode**: تم تاریک و روشن با ذخیره‌سازی
- **آیکون‌های زیبا**: هر ابزار با ایموجی مخصوص

### 📱 نحوه نصب

#### Android
1. سایت را در Chrome باز کنید
2. پیغام "نصب اپلیکیشن" را ببینید و روی "نصب" کلیک کنید
3. یا از منوی Chrome (⋮) گزینه "Add to Home screen" را انتخاب کنید

#### iOS (iPhone/iPad)
1. سایت را در Safari باز کنید
2. روی دکمه Share (مربع با فلش به بالا) کلیک کنید
3. به پایین اسکرول کنید و "Add to Home Screen" را انتخاب کنید
4. روی "Add" کلیک کنید

#### دسکتاپ (Chrome/Edge)
1. سایت را در مرورگر باز کنید
2. نماد نصب (➕) در نوار آدرس را ببینید
3. روی آن کلیک کنید و "Install" را انتخاب کنید
4. یا پیغام "نصب اپلیکیشن" در پایین صفحه را ببینید

## 🚀 استفاده

### جستجوی ابزار
- در کادر جستجو تایپ کنید
- با کلیدهای `/` یا `Ctrl+K` / `Cmd+K` به جستجو بروید
- جستجو در نام‌ها و توضیحات فارسی و انگلیسی

### تغییر تم
- دکمه 🌙/☀️ در بالای صفحه
- انتخاب شما ذخیره می‌شود

### کیبورد شورتکات
- `/` یا `Ctrl+K`: فوکوس به جستجو
- `Tab`: حرکت بین ابزارها
- `Enter`: باز کردن ابزار

## 💻 توسعه

### فایل‌های اصلی

## 🌟 ویژگی‌های فنی PWA

### Service Worker
- کش کردن فایل‌های استاتیک
- کار آفلاین
- بارگذاری سریع
- به‌روزرسانی خودکار

- نام و توضیحات دو زبانه
- آیکون‌های مختلف اندازه
- رنگ تم قابل تنظیم
- حالت Standalone (بدون نوار مرورگر)

### قابلیت‌های موبایل
- Touch-friendly UI
- Responsive Design
- Swipe و Gesture Support
- Native App Feel
