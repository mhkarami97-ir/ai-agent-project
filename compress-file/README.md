### نصب محلی

## 📦 نصب و راه‌اندازی

- 🔔 **اعلان‌های کاربری**: بازخورد بصری برای عملیات
- 💾 **ذخیره خودکار**: وضعیت برنامه به صورت خودکار ذخیره می‌شود
- 🇮🇷 **پشتیبانی کامل از فارسی**: رابط راست‌چین با فونت وضوح‌متن
- 📱 **طراحی واکنش‌گرا**: کاملاً سازگار با موبایل، تبلت و دسکتاپ
- 🌙 **تم تیره مدرن**: رابط کاربری زیبا با رنگ‌های مناسب برای چشم
### طراحی و رابط کاربری

- ⚙️ **اندازه تورفتگی**: تعداد فاصله‌ها برای تورفتگی را تنظیم کنید (2-8)
- ⚙️ **حذف فضای خالی**: فضاهای اضافی را برای کاهش حجم حذف کنید
- ⚙️ **حذف کامنت‌ها**: کامنت‌های کد را در فشرده‌سازی حذف کنید
### تنظیمات سفارشی

- 🗑️ **پاک کردن**: محتوای ورودی و خروجی را پاک کنید
- 📋 **کپی به کلیپ‌بورد**: نتیجه را با یک کلیک کپی کنید
- 📥 **دانلود نتیجه**: خروجی را به عنوان فایل دانلود کنید
- 📂 **بارگذاری فایل**: فایل‌های HTML، CSS و JS را بارگذاری کنید
- 📋 **چسباندن از کلیپ‌بورد**: کد را مستقیماً از کلیپ‌بورد وارد کنید
### عملیات ورودی/خروجی

- **آمار زنده**: نمایش حجم، تعداد خطوط و درصد کاهش حجم
- **ویرایشگر دو بخشی**: ورودی و خروجی به صورت همزمان نمایش داده می‌شوند
- **پشتیبانی از سه زبان**: HTML، CSS و JavaScript
- **فشرده‌سازی کد**: حجم فایل‌ها را با حذف فضای خالی و کامنت‌ها کاهش می‌دهد
- **فرمت‌دهی کد**: کدهای نامرتب را با تورفتگی مناسب و خوانایی بالا مرتب می‌کند
