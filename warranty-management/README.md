# Warranty Management

این پروژه یک اپ ساده‌ی مدیریت ضمانت‌نامه ارائه می‌دهد که کاملاً سمت کلاینت اجرا شده و بدون بک‌اند عمل می‌کند. اطلاعات در مرورگر ذخیره می‌شود، ورود کالا، تاریخ خرید و مدت گارانتی ثبت می‌شوند، و لیستی از کالای ثبت‌شده همراه با گزینه‌های ویرایش و حذف نمایش داده می‌شود. یادآوری نزدیک به پایان گارانتی نیز در بالای صفحه قابل‌مشاهده است.

## ساختار
- `index.html`: اسکلت RTL صفحه و المان‌های فرم و لیست را دارد.
- `assets/css/style.css`: استایل‌های واکنش‌گرا، تم روشن، استفاده از فونت کلاینتی و چینش مناسب موبایل را شامل می‌شود.
- `assets/js/app.js`: منطق خواندن/نوشتن در `localStorage`، و رندر لیست و کارت‌ها با قابلیت ویرایش و حذف.

## اجرا
1. مخزن را در شاخه‌ی دلخواه کلون یا کپی کنید.
2. روی فایل `index.html` در مرورگر باز کنید؛ نیازی به سرور نیست.
3. وارد کردن اطلاعات و ذخیره‌ی آن‌ها در مرورگر از طریق فرم انجام می‌شود.

## GitHub Pages
- شاخه‌ی پیش‌فرض برای هاست: `main` (یا `docs` اگر تنظیم کرده‌اید). فایل `index.html` روت است.
- فقط کافی‌ست مخزن را به GitHub پوش کنید و Pages را فعال کنید.

رابط کاربری کاملاً راست‌چین طراحی شده، از فونت Vazirmtn بهره می‌برد و برای نمایش در موبایل هم بهینه است.
