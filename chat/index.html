⚙️ تنظیمات اولیه

برای استفاده از این چت، نیاز به یک GitHub Personal Access Token دارید:

  1. به GitHub Settings → Developer settings → Personal access tokens بروید
  2. یک Token جدید با دسترسی gist ایجاد کنید
  3. Token را در فیلد زیر وارد کنید (فقط یک بار نیاز است)

⚠️ Token شما فقط در مرورگر شما ذخیره می‌شود و به سرور ارسال نمی‌شود
