# MetaClean

ابزار تک‌صفحه‌ای برای بررسی و حذف متادیتای تصاویر که کاملاً در مرورگر اجرا می‌شود و آماده‌ی انتشار روی GitHub Pages است.

## ویژگی‌ها
- رابط کاربری RTL و واکنش‌گرا با فونت Vazirmatn روی همه‌ی عناصر.
- پشتیبانی از کشیدن و رها کردن یا انتخاب فایل برای بارگذاری تصویر.
- نمایش متادیتای اصلی تصویر با استفاده از کتابخانه‌ی `exifr` بدون نیاز به سرویس سمت سرور.
- امکان تولید نسخه‌ی جدید از تصویر بدون متادیتا و دانلود مستقیم آن.
- ذخیره‌ی آخرین فایل پردازش‌شده در `localStorage` برای ارائه‌ی پیام راهنما به کاربر.
- اجرای ۱۰۰٪ سمت مرورگر و قابل میزبانی روی GitHub Pages یا هر هاست استاتیک مشابه.

## ساختار پروژه
```
meta-data/
├─ index.html
├─ assets/
│  ├─ css/
│  │  └─ styles.css
│  └─ js/
│     └─ app.js
└─ README.md
```

## نحوه‌ی استفاده
1. مخزن را کلون یا دانلود کنید.
2. پوشه را در یک وب‌سرور استاتیک (مانند `python -m http.server`) اجرا کنید یا آن را روی GitHub Pages منتشر نمایید.
3. فایل تصویر را در بخش بارگذاری بکشید یا دکمه‌ی انتخاب فایل را فشار دهید.
4. پس از نمایش متادیتا، در صورت تمایل دکمه‌ی «حذف متادیتا و ساخت نسخه جدید» را بزنید.
5. لینک دانلود ظاهر می‌شود و می‌توانید نسخه‌ی بدون متادیتا را دریافت کنید.

## توسعه و اجرای محلی
برای اجرای سریع محلی در ویندوز:
```powershell
cd meta-data
python -m http.server 8000
```
سپس مرورگر را باز کرده و به آدرس <http://localhost:8000> بروید.

## انتشار روی GitHub Pages
1. سورس را به یک مخزن GitHub پوش کنید.
2. از بخش **Settings → Pages** شاخه‌ی `main` و ریشه‌ی مخزن را انتخاب کنید.
3. پس از چند دقیقه آدرس صفحه فعال خواهد شد و وب‌اپ بدون نیاز به تنظیم اضافی اجرا می‌شود.

## ملاحظات امنیتی
- تمام پردازش‌ها روی دستگاه کاربر انجام می‌شود و فایلی به خارج ارسال نمی‌گردد.
- برای پاک‌سازی متادیتا، تصویر روی یک `<canvas>` رندر شده و خروجی جدیدی تولید می‌شود که فاقد داده‌های EXIF است.

## وابستگی‌ها
- فونت Vazirmatn از Google Fonts بارگذاری می‌شود.
- کتابخانه‌ی `exifr` از CDN جهت استخراج متادیتا استفاده می‌شود.
