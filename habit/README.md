# Habit Tracker

یک نرم‌افزار ردیابی عادت‌روزانه تمیز، طراحی‌شده برای صفحه‌‌های GitHub Pages.

## ساختار پروژه
- `index.html` – نقطه ورود، جهت‌دار به راست، شامل فرم ثبت عادت، جدول خلاصه، لیست عادت‌ها و نمودار پیشرفت.
- `assets/css/styles.css` – تمام قواعد بصری، طرح‌بندی واکنش‌گرا، کارت‌ها و دکمه‌ها.
- `assets/js/app.js` – مدیریت داده‌ها، ذخیره در `localStorage`, به‌روزرسانی آمار، رسم نمودار و یادآوری‌های زمان‌بندی‌شده.

## راه‌اندازی
1. مخزن را کلون کنید و وارد شاخه شوید.
2. فایل‌ها را روی GitHub Pages منتشر کنید (فقط پوشه اصلی کافی است).
3. مرورگر را باز کرده و روی `index.html` کلیک کنید یا آدرس GitHub Pages را باز کنید.

## نحوه استفاده
1. اطلاعات عادت را در فرم ثبت کنید؛ هدف روزانه و زمان یادآوری را تعریف نمایید.
2. با فشردن دکمه `ثبت تکمیل`, ثبت تکمیل روزانه انجام می‌شود و نمودار بروز می‌شود.
3. تعداد یادآورها، عادت‌های فعال، و بهترین استرک در بالای صفحه نمایش داده می‌شوند.
4. یادآورها به صورت خودکار بر اساس زمان واردشده در فرم نمایش پیدا می‌کنند و در بخش یادآورها فهرست می‌شوند.

## موارد رایج
- داده‌ها در `localStorage` نگهداری می‌شوند که مرورگر پشتیبانی می‌کند.
- هنگام حذف عادت، داده‌های آن نیز پاک و نمودار به‌روزرسانی می‌شود.
- نمودار هفتگی نشان می‌دهد هر روز چند تکمیل ثبت شده است.
