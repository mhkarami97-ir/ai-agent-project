# ریجکس پلی‌گراند - آزمایشگاه عبارات منظم

یک ابزار قدرتمند و کاربردی برای تست، آزمایش و یادگیری عبارات منظم (Regular Expressions) با رابط کاربری فارسی و طراحی واکنش‌گرا.

## 🌟 ویژگی‌های کلیدی

### ⚡ تست لحظه‌ای
- تست خودکار عبارات منظم هنگام تایپ
- نمایش فوری نتایج مطابقت‌ها
- برجسته‌سازی متن‌های منطبق

### 🔍 نمایش جزئیات
- نمایش تعداد کل مطابقت‌ها
- موقعیت دقیق هر مطابقت در متن
- نمایش گروه‌های capture شده
- توضیح ساده عبارت منظم

### 🎛️ کنترل پرچم‌ها (Flags)
- **g (Global)**: جستجوی تمام مطابقت‌ها
- **i (Case Insensitive)**: نادیده‌گیری حروف بزرگ/کوچک
- **m (Multiline)**: حالت چند خطی

### 💾 مدیریت Snippets
- ذخیره عبارات منظم پرکاربرد
- اضافه کردن نام و توضیحات برای هر snippet
- استفاده سریع از snippets ذخیره شده
- حذف snippets غیرضروری

### 🚀 الگوهای آماده
مجموعه‌ای از الگوهای پرکاربرد شامل:
- تشخیص اعداد
- حروف فارسی و انگلیسی
- آدرس ایمیل
- شماره موبایل ایران
- و موارد بیشتر...

## 🛠️ تکنولوژی‌ها

- **HTML5**: ساختار صفحه
- **CSS3**: طراحی واکنش‌گرا با Flexbox و Grid
- **JavaScript (ES6+)**: منطق اصلی برنامه
- **LocalStorage**: ذخیره‌سازی snippets در مرورگر
- **فونت Vazirmatn**: پشتیبانی کامل از زبان فارسی

## 📱 پشتیبانی از دستگاه‌ها

- **دسکتاپ**: تجربه کامل با تمام امکانات
- **تبلت**: رابط کاربری بهینه‌شده
- **موبایل**: طراحی واکنش‌گرا و کاربرپسند

## 🎯 مخاطبان هدف

- برنامه‌نویسان و توسعه‌دهندگان
- دانشجویان علوم کامپیوتر
- تحلیلگران داده
- همه علاقه‌مندان به یادگیری regex

## 🚀 نحوه استفاده

### راه‌اندازی محلی

1. فایل‌های پروژه را دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید
3. شروع به تست عبارات منظم کنید!

### استقرار روی GitHub Pages

1. repository را fork کنید
2. در تنظیمات repository، GitHub Pages را فعال کنید
3. از branch `main` استفاده کنید
4. لینک سایت شما: `https://[username].github.io/regex-playground`

## 📖 راهنمای استفاده

### تست عبارت منظم
1. عبارت منظم مورد نظر را در فیلد "عبارت منظم" وارد کنید
2. متن آزمایشی را در ناحیه "متن آزمایشی" بنویسید
3. روی دکمه "تست کن" کلیک کنید یا منتظر تست خودکار باشید

### ذخیره Snippet
1. عبارت منظم دلخواه را تست کنید
2. روی دکمه "ذخیره Snippet" کلیک کنید
3. نام و توضیحات اختیاری را وارد کنید
4. روی "ذخیره" کلیک کنید

### استفاده از الگوهای آماده
- روی هر یک از دکمه‌های "الگوهای آماده" کلیک کنید
- الگو به طور خودکار در فیلد عبارت منظم قرار می‌گیرد

## 🌐 ویژگی‌های فارسی

- **RTL Layout**: طراحی راست به چپ
- **فونت فارسی**: استفاده از Vazirmatn
- **محتوای فارسی**: تمام متون به زبان فارسی
- **پشتیبانی Unicode**: تشخیص حروف فارسی در regex
