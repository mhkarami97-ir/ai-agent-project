## 📝 نکات مهم

1. **پشتیبان‌گیری**: داده‌ها در مرورگر ذخیره می‌شوند، پس از پاک کردن cache مراقب باشید
2. **به‌روزرسانی**: برنامه خودکار به‌روز نمی‌شود، برای نسخه‌های جدید باید فایل‌ها را جایگزین کنید
3. **اینترنت**: فقط برای بارگذاری اولیه فونت نیاز است
4. **مرورگر خصوصی**: در حالت خصوصی مرورگر، داده‌ها ذخیره نمی‌شوند
