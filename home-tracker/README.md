## ✨ ویژگی‌های اصلی

### 📦 مدیریت اقلام
- افزودن اقلام با اطلاعات کامل (نام، دسته‌بندی، مقدار، تاریخ انقضا، محل قرارگیری)
- ویرایش و حذف اقلام موجود
- دسته‌بندی‌های متنوع (لبنیات، گوشت، سبزیجات، میوه‌جات، غلات، کنسرو، نوشیدنی، تنقلات، لوازم نظافت، لوازم بهداشتی و...)
- محل‌های مختلف نگهداری (یخچال، فریزر، انباری، کابینت)

### ⏰ مدیریت تاریخ انقضا
- محاسبه خودکار روزهای باقی‌مانده تا انقضا
- وضعیت رنگی اقلام:
  - 🟢 **تازه**: بیش از 7 روز تا انقضا
  - 🟡 **در حال انقضا**: کمتر از 7 روز تا انقضا
  - 🔴 **منقضی شده**: تاریخ انقضا گذشته
- هشدارهای هوشمند در بالای صفحه
- به‌روزرسانی خودکار هر 5 دقیقه

### 📊 آمارها و گزارش‌ها
- نمایش کل اقلام موجود
- تعداد اقلام تازه
- تعداد اقلام در حال انقضا
- تعداد اقلام منقضی شده
- کارت‌های آماری با رنگ‌بندی مناسب

### 🔍 جستجو و فیلتر
- جستجوی سریع بر اساس نام قلم
- فیلتر بر اساس وضعیت (همه، تازه، در حال انقضا، منقضی شده)
- مرتب‌سازی بر اساس:
  - نزدیک‌ترین انقضا
  - دورترین انقضا
  - نام (الفبا)
  - تاریخ افزودن
- میانبر صفحه‌کلید: `Ctrl/Cmd + K` برای فوکوس روی جستجو

### 💾 ذخیره‌سازی
- استفاده از IndexedDB مرورگر
- ذخیره‌سازی دائمی و امن
- بدون نیاز به اتصال اینترنت پس از بارگذاری اولیه
- عدم نیاز به سرور یا بکند

### 🎨 طراحی و رابط کاربری
- طراحی کاملاً ریسپانسیو (موبایل، تبلت، دسکتاپ)
- پشتیبانی کامل از RTL
- انیمیشن‌های نرم و حرفه‌ای
- رنگ‌بندی زیبا و چشم‌نواز
- آیکون‌های Emoji برای نمایش بهتر
- حالت تاریک و روشن

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- مرورگر مدرن با پشتیبانی از IndexedDB (Chrome, Firefox, Safari, Edge)
- هیچ سرور یا دیتابیس خارجی نیاز نیست

### روش اول: اجرای محلی

1. فایل‌های پروژه را دانلود کنید
2. فایل `index.html` را با مرورگر باز کنید
3. شروع به استفاده کنید!

### روش دوم: GitHub Pages

1. پروژه را Fork کنید یا به repository خود push کنید
2. به تنظیمات repository بروید
3. بخش **Pages** را انتخاب کنید
4. Branch مورد نظر (معمولاً `main` یا `master`) را انتخاب کنید
5. پوشه Root را انتخاب کنید
6. روی Save کلیک کنید
7. پس از چند دقیقه، سایت شما در آدرس زیر در دسترس خواهد بود:
   ```
   https://[username].github.io/[repository-name]
   ```

## 🎯 نحوه استفاده

### افزودن قلم جدید
1. فرم "افزودن قلم جدید" را پر کنید
2. فیلدهای ضروری:
   - نام قلم
   - دسته‌بندی
   - تعداد/مقدار
   - تاریخ انقضا
3. فیلدهای اختیاری:
   - محل قرارگیری (پیش‌فرض: یخچال)
   - یادداشت
4. روی دکمه "ذخیره" کلیک کنید

### ویرایش قلم
1. روی دکمه "ویرایش" قلم مورد نظر کلیک کنید
2. اطلاعات را ویرایش کنید
3. روی "ذخیره تغییرات" کلیک کنید
4. برای انصراف، روی "انصراف" یا `ESC` بزنید

### حذف قلم
1. روی دکمه "حذف" قلم مورد نظر کلیک کنید
2. حذف را تأیید کنید

### جستجو و فیلتر
- در کادر جستجو تایپ کنید تا نتایج فیلتر شوند
- روی دکمه‌های فیلتر (همه، تازه، در حال انقضا، منقضی شده) کلیک کنید
- از منوی مرتب‌سازی برای تغییر ترتیب نمایش استفاده کنید

### حذف همه اقلام
- روی دکمه "حذف همه" در بالای لیست اقلام کلیک کنید
- حذف را تأیید کنید (این عمل قابل بازگشت نیست!)

## 🔒 امنیت و حریم خصوصی

- تمام داده‌ها به صورت محلی در مرورگر شما ذخیره می‌شوند
- هیچ اطلاعاتی به سرور ارسال نمی‌شود
- داده‌ها فقط در دستگاه شما قابل دسترسی هستند
- پاک کردن cache مرورگر، داده‌ها را حذف می‌کند

## 💡 نکات و ترفندها

### میانبرهای صفحه‌کلید
- `Ctrl/Cmd + K`: فوکوس روی جستجو
- `ESC`: بستن مودال ویرایش

### بهترین شیوه‌ها
- تاریخ انقضا را دقیق وارد کنید
- از یادداشت‌ها برای توضیحات اضافی استفاده کنید
- به طور منظم اقلام منقضی شده را حذف کنید
- هشدارها را جدی بگیرید و اقلام در حال انقضا را زودتر مصرف کنید

### پشتیبان‌گیری
چون داده‌ها در مرورگر ذخیره می‌شوند:
- از پاک کردن cache مرورگر مراقب باشید
- می‌توانید از Developer Tools مرورگر برای خروجی گرفتن از IndexedDB استفاده کنید
- برای استفاده در دستگاه‌های مختلف، اطلاعات را دوباره وارد کنید

## 🐛 رفع مشکلات رایج

### داده‌ها ذخیره نمی‌شوند
- مطمئن شوید که حالت خصوصی (Private/Incognito) مرورگر را استفاده نمی‌کنید
- بررسی کنید که مرورگر شما از IndexedDB پشتیبانی می‌کند
- دسترسی‌های لازم را به مرورگر بدهید

### صفحه به درستی نمایش داده نمی‌شود
- Cache مرورگر را پاک کنید و دوباره بارگذاری کنید
- مطمئن شوید که فایل‌های CSS و JS به درستی بارگذاری شده‌اند
- از مرورگر به‌روز استفاده کنید

### فونت فارسی نمایش داده نمی‌شود
- مطمئن شوید که اتصال اینترنت فعال است (برای بارگذاری اولیه فونت)
- در صورت بارگذاری کامل، فونت به صورت cache شده در مرورگر می‌ماند
