# Home Inventory Tracker 🏠

A comprehensive Persian (Farsi) web application for managing home inventory with expiration date tracking and smart alerts.

[فارسی](README.md) | **English**

## Features

- ✅ **Item Management**: Add, edit, and delete items with full details
- ✅ **Expiration Tracking**: Automatic calculation of days until expiration
- ✅ **Smart Alerts**: Get notified about expiring and expired items
- ✅ **Search & Filter**: Quick search and multiple filter options
- ✅ **Statistics Dashboard**: Visual overview of your inventory
- ✅ **Offline Storage**: Uses IndexedDB for persistent browser storage
- ✅ **Fully Responsive**: Works perfectly on mobile, tablet, and desktop
- ✅ **RTL Support**: Full right-to-left layout for Persian language
- ✅ **No Backend Required**: Pure client-side application

## Technology Stack

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript (ES6+)
- IndexedDB for data persistence
- Vazirmatn font family

## Quick Start

### Local Usage
1. Clone or download this repository
2. Open `index.html` in your browser
3. Start adding items!

### GitHub Pages Deployment
1. Fork this repository
2. Go to Settings → Pages
3. Select your branch (main/master) and root folder
4. Save and wait for deployment
5. Access your site at `https://[username].github.io/[repo-name]`
